# 料金入力補助機能基本設定

料金入力補助機能基本設定

【設定】＞ 【基本設定】＞【料金入力補助機能基本設定】
料金入力補助機能基本設定は、料金ランクに料金入力補助機能のルール付けを行います。

![料金入力補助機能基本設定_R.PNG](https://www.temairazu.com/net/manual/img/料金入力補助機能基本設定_R.PNG)

①基本となる料金ランクを選択します。
②「保存」をクリックします。
③設定した基本ランクにプラス（マイナス）何円（％）したランクにするか入力し「設定する」をクリックします。
※すべての料金ランクに補正の料金を入力してください。
※基本の料金ランクより低くする場合については｢-｣をつけて料金を入力してください。
④「設定する」をクリックします。

料金入力補助機能基本設定を設定後、プラン料金設定時に料金簡単入力機能を利用できます。
