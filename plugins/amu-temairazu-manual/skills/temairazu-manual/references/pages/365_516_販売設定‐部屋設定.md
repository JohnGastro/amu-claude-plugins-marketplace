販売設定‐部屋設定

【設定】＞【予約サイト設定】＞【管理画面】（ANA Universal MaaS）＞【販売設定】＞【部屋設定】
【オプション】＞【ANA Universal MaaS】＞【販売設定】＞【部屋設定】

![ANA部屋設定102_R.png](https://www.temairazu.com/net/manual/img/ANA部屋設定102_R.png)

①「追加」から部屋情報の登録ができます。
②「部屋情報の一括登録」からExcelファイルのアップロードによって、部屋情報を一括で登録できます。
　 ※部屋情報の一括ファイル取り込みでは新規登録にのみ対応しています。
　 既存の部屋情報の二重登録にご注意ください。
③作成した部屋のコピーが可能です。
　 ※登録後は⑨「各種設定の最新化」をクリックしてください。
　 ※⑨をクリックした段階で設定した情報が手間なしNEXTへ反映されます。
　　（「登録」をクリックした時点では反映されません）

「追加」から部屋情報の登録

![ANA部屋編集02_RR.png](https://www.temairazu.com/net/manual/img/ANA部屋編集02_RR.png)

①-1 紐付けるTEMAIRAZUの部屋マスタを選択します。部屋マスタの詳細についてはこちら
また販売先の選択については「全て」または「個別」選択時ANA Universal MaaSにチェックされていることを確認してください。

①-2 設定完了し販売できる状態になってから販売中を選択してください。
　　　 （新規作成の時点では準備中の選択を推奨）

①-3 サイト側に掲載される部屋名や部屋に関する情報、広さを入力します。
　　　 ※手間なしNEXTのデザイン適用版をご利用の場合、「部屋販売名」および「部屋詳細」について、
　　　 日本語欄に設定した内容が自動でGoogle翻訳され、多言語表示されます。英語欄は利用できません。

①-4 部屋の広さを半角数字で入力します。

①-5 部屋カテゴリを選択します。

①-6 部屋タイプを選択します。

①-7 ANA Universal MaaSでは使用しないため操作不要です。
　　　 （プラン一括管理で利用する機能となります。詳細はこちら）

①-8 宿泊可能人数の最小・最大を設定します。
　　　 ※大人＋寝具の必要な子供(子供A～子供C)の人数設定になります。
　　　 ※設定後に内容を変更した場合は料金も修正してください。

①-9 寝具を必要としない子供について宿泊可能な人数を入力してください。
　　　設定しない場合は0を入力してください。

①-10 有効開口幅(通過できる幅)をご記入ください。

①-11 床からの高さをご記入ください。

①-12 壁や配置している家具からベッドまでの距離が分かるようにご記入ください。

①-13 有効開口幅(通過できる幅)をご記入ください。

①-14 トイレの広さ等のバリアフリー情報が分かる写真をご登録ください。

①-15 洗面台のシンクしたのスペースの有無が分かる写真をご登録ください。

①-16 シャワーの形状、高さ、浴室内車いすの設置、浴室内に滑り止めがあるか、手すりがあるか、移乗して座るスペースがあるかなどが分かるように画像をご登録ください。

①-17 「登録」をクリックすると、「登録しました。」と表示され、部屋一覧画面に遷移します。
　　　　※登録後は⑨「各種設定の最新化」をクリックしてください。
　※⑨をクリックした段階で設定した情報がANA Universal MaaSへ反映されます。
　（①-17の時点では反映されません）

「部屋情報の一括登録」から部屋情報の登録

![新UIExcel取り込み1_R.png](https://www.temairazu.com/net/manual/img/新UIExcel取り込み1_R.png)

②-a 「ダウンロード」ボタンをクリックしてExcelファイルをダウンロードしたあと、
　　　 一括登録するデータを入力してファイルをアップロードしてください。
　　　 ※ダウンロードしたExcelファイルの「注意事項」を確認して一括登録するデータを入力してください。

②-b Excelファイルを選択してください。
　　　 ※10MBを超えるファイルはアップロードできないため、ファイルを分割してください。

②-c ②-bで選択したExcelファイルをアップロードします。
※アップロード後は⑨「各種設定の最新化」をクリックしてください。
※⑨の段階で設定した情報がANA Universal MaaSへ反映されます。
　（②-cの時点では反映されません）

部屋作成後の部屋情報の編集など

![ANA部屋編集304_R.png](https://www.temairazu.com/net/manual/img/ANA部屋編集304_R.png)

④作成した部屋名をクリックすると、部屋情報の編集や画像登録ができます。

⑤部屋毎・子供区分毎に宿泊可能人数（定員）に含めるかの設定をすることができます。
※添い寝のお子様を宿泊可能人数（定員）に含めずに受け付けたい等の場合にご設定ください。

（例）
部屋の設定は2名定員だが、添い寝のお子様を1名受け入れ可とし、
大人2名・添い寝の子供1名で合計3名の予約も取れるようにしたい。

※「含める」を選択した子供区分の人数は、宿泊可能人数（定員）にカウントされます。
　 例：大人2名＋添い寝の子供1名（カウントする）＝ 3名のため、2名定員の部屋では予約不可。

※「含めない」を選択した子供区分の人数は、宿泊可能人数（定員）のカウントから除外されます。
　 例：大人2名＋添い寝の子供1名（カウントしない）＝ カウント上は2名のため、2名定員の部屋で予約可。

　 「含めない」で設定した子供の受入人数は ⑥「宿泊可能人数に含めない子供 最大受入人数」 で
　 制限をかけることができます。

※初期設定では、寝具なしの子供（子供D・E）は宿泊可能人数（定員）に含まれません。
　 子供D: 食事なし・寝具なし
　 子供E: 食事あり・寝具なし

⑥ ⑤で宿泊可能人数に含めない子供の設定を行っている場合、受け入れられる最大人数を設定してください。
※子供設定がない場合は設定不要です。

⑦部屋を削除または一括削除ができます。

⑧「登録・更新」をクリックしてください。

⑨「各種設定の最新化」をクリックしてください。
※⑨の段階で設定した情報がANA Universal MaaSへ反映されます。
　（⑧の時点では反映されません）
