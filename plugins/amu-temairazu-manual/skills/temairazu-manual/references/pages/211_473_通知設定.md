# 通知設定

> id=473 ／ order=211 ／ 出典: 手間いらず Webマニュアル

通知設定

【ツール】＞【各種ツール】＞【ベストレートチェッカー設定】＞【通知設定】

翌々月末日までを対象として料金比較を行い、
各予約サイトの料金よりも自社宿泊予約システムの料金が高い場合に、
1日1回、在庫コントロール画面へのアラート表示およびメール通知を行うことができます。

![通知設定_R.png](https://www.temairazu.com/net/manual/img/通知設定_R.png)

①在庫コントロール画面に、以下のアラートを表示するかどうかを選択してください。

②メール通知を送信するかどうかを選択してください。
　送信先メールアドレスは、カンマ（,)で区切ることで複数設定できます。
　※「no-reply@temairazu.com」からのメールを受信できるよう、あらかじめ受信設定を行ってください。

③「登録する」をクリックしてください。
