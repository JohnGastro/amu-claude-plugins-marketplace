# 料金一覧

> id=163 ／ order=116 ／ 出典: 手間いらず Webマニュアル

料金一覧

【ツール】＞【料金一覧】

月ごとに予約サイトと部屋タイプを絞って、プランに設定した料金を一覧で表示することができます。

![新料金一覧_R.png](https://www.temairazu.com/net/manual/img/新料金一覧_R.png)

①予約サイトを選択します。
※予約サイトの選択肢の並べ替えは下記から変更が可能です。
【設定】＞【基本設定】＞【料金一覧予約サイト並べ替え設定】

②部屋マスタを選択します。
③プランを選択します。
④料金確認したい期間を指定します。
⑤「検索」をクリックします。

![新料金一覧検索結果_R.png](https://www.temairazu.com/net/manual/img/新料金一覧検索結果_R.png)

⑥再度条件を変更して検索したい場合にクリックします。
⑦ ①～④で指定した条件のプラン料金一覧が表示されます。
　※手仕舞いをかけている場合は「０」、料金設定・料金ランク未設定の場合「-(ハイフン)」表記となります。
⑧表示された料金一覧をExcelダウンロードできます。
