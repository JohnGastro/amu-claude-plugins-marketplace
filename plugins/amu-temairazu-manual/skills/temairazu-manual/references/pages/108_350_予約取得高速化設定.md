# 予約取得高速化設定

> id=350 ／ order=108 ／ 出典: 手間いらず Webマニュアル

予約取得高速化設定

【設定】＞【基本設定】＞【予約取得高速化設定】

＜対象予約サイト＞

じゃらん 
一休.com（予約データダウンロードサービスのご契約が必須となります）

本機能を有効にすると、予約が発生した際に予約サイトから送られる予約通知メールを手間いらずで受信（※）し、
即座に予約取得を開始することで従来よりも早く予約情報を取得することが可能となります。

※手間いらずが受信した予約通知メールをご指定のメールアドレスに転送する設定も可能です。

![予約取得高速化設定.png](https://www.temairazu.com/net/manual/img/予約取得高速化設定.png)

＜設定手順＞

①設定する予約サイトを選択してください。

②予約取得設定を「利用する」に変更してください。

③サイトからの予約通知メールを転送でお受け取りになりたい場合は「利用する」を選択してください。
　 ※じゃらんは設定を行う際に転送が必要となります。初回設定時は必ず「利用する」を選択してください。

④転送先メールアドレスにご施設様のメールアドレスを入力してください。
　 ※手間いらずに受信した予約通知メールを、こちらで設定したメールアドレスに転送します。
　 ※迷惑メールに振り分けられる場合は「info@ar-tm.com」からのメールを受信可能にしてください。

⑤「登録」をクリックし、設定を保存してください。

⑥「メールテスト」をクリックし、④で入力したメールアドレスにテストメールが受信されるか確認してください。

⑦表示されているメールアドレスをコピーしてください。

⑧設定する予約サイトの管理画面にログインし、予約通知メールの設定画面で⑦でコピーしたメールアドレスを
　 宛先に設定してください。

＜各サイトの設定方法＞

■じゃらん管理画面での設定方法
1. 基本情報設定＞予約基本設定（通知管理）をクリックしてください。

（通知管理の設定＞予約通知方法が「FAX」選択中の場合）
2. 予約通知方法＞携帯メールを「受け取る」選択してください。
3. 通知管理の設定＞連絡先２携帯メールアドレスに⑦でコピーしたメールアドレスを設定してください。
4. 「送信テスト（パスワード取得）」をクリックすると、④で設定したアドレス宛てに設定変更用のパスワードが届きます。
5. 届いたパスワードを「パスワード」枠内に貼り付けてください。
6. 画面下部の「設定情報を確認する」から画面を進め、設定変更を完了してください。

（通知管理の設定＞予約通知方法が「メール」選択中の場合）
2. 通知管理の設定＞連絡先１PCメールアドレスに⑦でコピーしたメールアドレスを設定してください。
　 ※既に連絡先１PCメールアドレスに別のアドレスが登録されている場合、連絡先２携帯メールアドレスに設定してください。
3. 「送信テスト（パスワード取得）」をクリックすると、④で設定したメールアドレス宛てに設定変更用のパスワードが届きます。
4. 届いたパスワードを「パスワード」枠内に貼り付けてください。
5. 画面下部の「設定情報を確認する」から画面を進め、設定変更を完了してください。

■一休.com管理画面での設定方法
予約データダウンロードサービスが有効な場合のみ本設定が可能です。一休.com管理画面「各種サービス申し込み」で
予約データダウンロードサービスが「利用可」と表示されているかを設定前に必ずご確認ください。

1. 基本設定＞施設情報＞通知情報をクリックしてください。
2. 「通知方法」 の「Mail」にチェックが入っていない場合はチェックを入れてください。
3. 「メールアドレス」 に、⑦でコピーしたメールアドレスを設定してください。
　　※既に別のメールアドレスが登録されている場合は「メールアドレスCc」 に設定してください。
4. 画面下部の「変更」から画面を進め、設定変更を完了してください。
