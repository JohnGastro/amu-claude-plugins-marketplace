# 設定一覧画面

> id=336 ／ order=45 ／ 出典: 手間いらず Webマニュアル

設定一覧画面

【設定】＞【基本設定】＞【コネクティングルーム設定】
【設定】＞【基本設定】＞【部屋マスタ】＞画面最下部の【コネクティングルーム設定】

![新UI_コネクティング02.png](https://www.temairazu.com/net/manual/img/新UI_コネクティング02.png)

①コネクティングルーム設定の新規登録ボタンです。

②コネクティングルームの名称が表示されます。クリックすると設定内容の編集ができます。

③コネクティングルームの部屋数が表示されます。

④コネクティングルーム設定後、紐付く部屋に既に予約が入っている場合や、在庫数の調整を行っている場合に、
　コネクティングルームの在庫を調整するボタンです。
　新規設定後、または設定を変更した際は必ず「調整」ボタンをクリックしてください。
　※コネクティングルームに既に予約が入っている場合や、在庫数の調整を行っている場合、
　　「調整」ボタンをクリックしても紐付く部屋の在庫数は自動では調整されません。
　　そのため、在庫コントロール画面にて、紐付く部屋の在庫数を手動で調整してください。

⑤コネクティングルームに紐付く部屋の名称が表示されます。

⑥コネクティングルームに紐付く部屋の部屋数が表示されます。

⑦手仕舞い制限在庫数が表示されます。詳細は設定の登録をご覧ください。

⑧コネクティングルーム設定の削除ができます。チェックボックスにチェックを入れて
　後述の⑨をクリックすることで、一括削除ができます。

⑨上記⑧で選択した設定を一括削除するためのボタンです。
