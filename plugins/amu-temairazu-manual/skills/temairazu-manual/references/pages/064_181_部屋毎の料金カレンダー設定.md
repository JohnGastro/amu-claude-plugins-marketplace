# 部屋毎の料金カレンダー設定

部屋毎の料金カレンダー設定

【在庫コントロール】＞各部屋マスタ名右の【>>】のボタン

部屋タイプごとに手仕舞いや割引・割増設定、料金カレンダーの設定ができます。
　※各予約サイトに更新を行うには、あらかじめ基本料金カレンダーが設定されていることが
　　前提となります。部屋別カレンダーのみ料金ランクの設定を行っても正常に料金更新が
　　されませんので、ご注意ください。

![部屋毎の料金カレンダー設定画像①.png](https://www.temairazu.com/net/manual/img/部屋毎の料金カレンダー設定画像①.png)

①「>>」をクリックすると、部屋別カレンダー画面が開きます。

![部屋毎の料金カレンダー設定画像②.png](https://www.temairazu.com/net/manual/img/部屋毎の料金カレンダー設定画像②.png)

基本的なランク選択などの設定方法は基本料金カレンダーと同じです。

②プルダウンで別の部屋のカレンダーに切り替えができます。
　　※選択中、未確定の変更内容は失われますのでご注意ください。

③左のカレンダーで日付を選択してから「手仕舞い」をクリックすると、手仕舞いを選択できます。
　手仕舞いを選択中、または設定済みの日付を選択し「手仕舞い設定解除」をクリックすると、
　手仕舞いを解除した状態になります。
　　※⑤「登録」ボタン⇒OKを押すまでは未確定です。

④左のカレンダーで日付を選択してから割引・割増率を設定して「入力する」をクリックすると、
　割引・割増設定を入力できます。割引・割増設定を入力中、または設定済みの日付を選択し
　「割引・割増設定解除」をクリックすると、割引・割増設定を解除した状態になります。
　　※⑤「登録」ボタン⇒OKを押すまでは未確定です。

⑤「登録」ボタン⇒OKを押すと、選択したランク、手仕舞い設定、割引・割増設定などが確定、
　反映されます。

複数部屋の一括設定
現在表示している部屋別カレンダー設定を他の部屋別カレンダーにコピーする機能です。

![部屋毎の料金カレンダー設定画像③02_R.png](https://www.temairazu.com/net/manual/img/部屋毎の料金カレンダー設定画像③02_R.png)

⑥手仕舞い、割引・割増設定、料金ランクの各設定をコピーするか、全てコピーするか選択できます。

⑦右側の部屋一覧でカレンダーのコピーを行いたい部屋名を選択して「<<」をクリックすると、
　左側の枠内に部屋名が移動します。右上の条件指定で、部屋一覧に表示される部屋を絞り込む
　ことができます。

⑧⑦で、選択中の登録先部屋欄に移動した部屋をまとめて保存したい場合は
　「グループセットの保存・呼出」をクリックし、グループセットの名称を入力のうえ、
　「選択中の部屋を新規登録(最大10セット)」にチェックし「登録」をクリックしてください。
　※部屋別料金カレンダー設定で複数部屋を一括登録する際にグループセット名を
　選択いただくだけで選択中の部屋グループの中に対象の部屋が選択されます。

⑨コピーを行いたい部屋を全て選択したら、「登録」ボタンをおします。OKを選択したら
　カレンダーのコピーは完了です。

予約紐付け解除設定

⑩この部屋に紐付けられた全期間の予約が紐付け解除されます。
　※この部屋に紐付いていた予約が全て部屋に紐付かない予約となり、予約されていた部屋数分の
　　戻り在庫が発生しますのでご注意ください。
