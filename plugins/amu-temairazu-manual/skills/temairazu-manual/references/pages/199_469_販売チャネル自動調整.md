# 販売チャネル自動調整

> id=469 ／ order=199 ／ 出典: 手間いらず Webマニュアル

販売チャネル自動調整

【ツール】＞【各種ツール】＞【販売チャネル自動調整】

対応バージョン：手間いらず自動

指定した期間内の残室数や稼働率に応じて、自動で手仕舞いを行うことができる機能です。

※事前に、自動料金設定期間一覧の設定を行ってください。

![販売チャネル自動作成機能1_R.png](https://www.temairazu.com/net/manual/img/販売チャネル自動作成機能1_R.png)

①実行期間・設定タイトル・実行条件を指定して検索できます。
②期間名をクリックすると、調整ツール期間設定・販売チャネル自動調整設定を設定できます。
③設定の利用有無をON／OFFで選択できます。
④詳細設定の登録・編集ができます。
⑤設定内容を即時適用する場合は、「設定反映」をクリックしてください。

■②調整ツール期間設定・販売チャネル自動調整設定 追加/編集画面

![販売チャネル自動作成機能2_R.png](https://www.temairazu.com/net/manual/img/販売チャネル自動作成機能2_R.png)

②-1 設定タイトルを入力してください。
②-2 対象サイトを選択してください。
②-3 実行条件を選択してください。
②-4 キャンセルにより残室数や稼働率が変動した場合、手仕舞いを解除するかどうかを選択してください。
②-5 登録した内容を削除できます。
②-6 「登録」をクリックしてください。

■④詳細設定 追加/編集画面

![販売チャネル自動作成機能3_R.png](https://www.temairazu.com/net/manual/img/販売チャネル自動作成機能3_R.png)

④-1 「追加」をクリックしてください。

![販売チャネル自動作成機能4_R.png](https://www.temairazu.com/net/manual/img/販売チャネル自動作成機能4_R.png)

④-2 部屋タイプを選択してください。
④-3 ④-2で「全部屋」を選択した場合、除外したい部屋タイプがあれば選択してください。
④-4 日数および稼働率／残室数を設定し、手仕舞いを行う場合はチェックを入れてください。
　　　 「＋」「－」ボタンをクリックすると、設定項目の追加・削除ができます。
④-5 「並べ替え」をクリックすると、日数・残室数・稼働率の順に自動で並び替えられます。
④-6 「登録」をクリックしてください。
