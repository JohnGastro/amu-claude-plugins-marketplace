# 部屋の取得と在庫情報の取得

> id=391 ／ order=372 ／ 出典: 手間いらず Webマニュアル

部屋の取得と在庫情報の取得

・部屋情報の取得

【設定】＞【旅行会社設定】＞【旅行会社一覧】＞各旅行会社の【部屋一覧】より
「旅行会社から部屋を取得する」をクリック

　　※取得が必要なタイミング
　　旅行会社側で新たに部屋を作成された場合、手間いらずに最新の部屋情報を取得する必要があります。

![旅行会社設定部屋一覧_R.png](https://www.temairazu.com/net/manual/img/旅行会社設定部屋一覧_R.png)

①「旅行会社から部屋を取得する」をクリックすると、上図のように旅行会社で販売している部屋情報が取得されます。
②「全期間在庫取得」をクリックすると当月を含む24ヶ月分の在庫情報を取得します。
※取得した在庫数は「在庫確認」よりご確認ください。

※初めて取得した部屋は、部屋情報取得後に必ず全期間の在庫情報を取得してください。
取得するまでは在庫の変更が行えないのでご注意ください。
