# 予約通信数確認

> id=89 ／ order=109 ／ 出典: 手間いらず Webマニュアル

予約通信数確認 

【ツール】＞【各種ツール】＞【予約通信数確認】

　選択した月の予約・変更・キャンセル情報と、合計通信数を表示します。
　※合計通信数は最下部にあります。

※当月は表示されません。前月までの通信数が表示されます。

![新予約通信数確認.png](https://www.temairazu.com/net/manual/img/新予約通信数確認.png)

①予約通信数を表示します。

②予約変更通信数を表示します。

③キャンセル予約の通信数を表示します。

④合計通信数が表示されます。

⑤表示された予約一覧のCSVをダウンロードできます。
※オプションのCSV出力機能をお申込みいただいている場合のみご利用いただけます。
