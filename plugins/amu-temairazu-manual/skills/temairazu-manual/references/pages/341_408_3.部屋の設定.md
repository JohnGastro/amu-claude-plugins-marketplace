# 3.部屋の設定

3.部屋の設定

スマートロックシステム連携で利用する部屋の設定を行います。部屋タイプではなく、
施設内の各お部屋の登録と設定を行います。

![SL連携02.png](https://www.temairazu.com/net/manual/img/SL連携02.png)

①登録した部屋ID・部屋名が表示されます。部屋名をクリックすると、部屋名の編集、部屋の削除が行えます。
　　※部屋IDは変更できません。

②ロックデバイスとのマッピング状況が表示されます。
　　※ロックデバイスとのマッピングは、別途ロックデバイスの設定とマッピング設定を行う必要があります。

③PIN発行の有無が表示されます。ロックデバイスとのマッピングがない部屋は「-」と表示されます。

④部屋情報の削除ができます。
　　※PINが発行されている部屋は削除できません。

⑤スマートロックシステム連携メニュー画面に戻ります。

⑥手動で部屋を登録できます。詳細は下記「手動で部屋を登録する」をご参照ください。

⑦CSVファイルで部屋を一括登録できます。詳細は下記「部屋を一括登録する」をご参照ください。

手動で部屋を登録する

![SL連携03.png](https://www.temairazu.com/net/manual/img/SL連携03.png)

①部屋ID、部屋名を入力します。
　　※部屋IDは半角英数のみご利用いただけます。
　　※部屋IDは登録後に変更できません。

②登録せずに前の画面に戻ります。

③部屋の登録をします。

部屋を一括登録する

CSVファイルの読み込みを行うことで、一括で部屋の追加登録・部屋名の変更ができます。（部屋の削除はできません）
後述の②でダウンロードしたCSVファイルを編集し、①の手順で読み込みを行ってください。

![SL連携04-2.png](https://www.temairazu.com/net/manual/img/SL連携04-2.png)

①編集したCSVファイルを選択して「選択したCSVの読み込みを行う」をクリックします。
　 下図のとおり部屋名の変更、追加した部屋が青文字で表示されますので、ご確認のうえ
　 問題がなければ「保存」ボタンを押してください。

![SL連携05.png](https://www.temairazu.com/net/manual/img/SL連携05.png)

②CSVファイルのダウンロードが行えます。ダウンロードしたファイルには作成済みの部屋（上図③）の情報が
　 予め入力されています。必要に応じて部屋名の変更、追加する部屋IDと部屋名を追加入力し、上書き保存して
　 ①の工程で読み込みを行ってください。

③作成済みの部屋が表示されます。

④前の画面に戻ります。
