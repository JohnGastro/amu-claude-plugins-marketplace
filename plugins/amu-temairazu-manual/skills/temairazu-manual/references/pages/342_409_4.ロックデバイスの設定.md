# 4.ロックデバイスの設定

4.ロックデバイスの設定

スマートロックシステムからロックデバイスの情報を取得することができます。
　※あらかじめスマートロックシステムでデバイスの登録をしておく必要があります。

![SL連携06b.png](https://www.temairazu.com/net/manual/img/SL連携06b.png)

①取得されているロックデバイスのID・名称が表示されます。

②部屋とのマッピング状況が表示されます。
　　※部屋とのマッピングは、別途部屋の設定とマッピング設定を行う必要があります。

③PIN発行の有無が表示されます。部屋とのマッピングがない部屋は「-」と表示されます。

④ロックデバイスの削除ができます。
　　※PINが発行されているロックデバイスは削除できません。

⑤スマートロックシステム連携メニュー画面に戻ります。

⑥連携しているスマートロックシステムからロックデバイスの情報を取得・登録します。
　 詳細は下記「ロックデバイスの情報取得」をご参照ください。
　　※あらかじめご利用されているスマートロックシステム側でデバイスの登録をしておく必要があります。

ロックデバイスの情報取得

下図青枠の箇所には、既に登録済みのロックデバイス情報が表示されます。

![SL連携09.png](https://www.temairazu.com/net/manual/img/SL連携09.png)

⑥-1 「スマートロックシステムからロックデバイスを取得する」ボタンをクリックしてください。

![SL連携10.png](https://www.temairazu.com/net/manual/img/SL連携10.png)

⑥-2 現在登録していないロックデバイスの情報を取得すると、青文字で表示されます。

⑥-3 「保存」をクリックしてください。
