# ゲストとのメッセージ交換機能

> id=550 ／ order=118 ／ 出典: 手間いらず Webマニュアル

ゲストとのメッセージ交換機能

【在庫コントロール画面】＞【

![メッセージ交換機能アイコン.png](https://www.temairazu.com/net/manual/img/メッセージ交換機能アイコン.png)

アイコン】

「ゲストとのメッセージ交換機能」とは、TEMAIRAZU画面上でゲスト(宿泊者)とメッセージのやりとりができる機能です。
TEMAIRAZU画面から送信したメッセージは、各予約サイトの管理画面上に反映します。
各予約サイトの管理画面から送信したメッセージもTEMAIRAZU上で確認することができます。

≪対象予約サイト≫
・Booking.com
・Airbnb
※Airbnbは手間いらずとのアカウント再紐付け操作を行う必要があります。
アカウント再紐付け操作に関しては手間いらずカスタマーサポートセンターまでご連絡ください。
・Trip.com（New）

はじめにアカウント設定画面でメッセージ機能を有効にしてください

![メッセージBcom.png](https://www.temairazu.com/net/manual/img/メッセージBcom.png)

設定＞予約サイト設定＞対象サイトのアカウント設定より
アカウント設定画面下部のメッセージ機能を「利用する」を選択し「設定を反映する」をクリックしてください。
※Trip.com（New）の場合、IMアカウント名の欄に
Trip.com(New)のeBookingのアカウント（eBooking user name）名も入力してください。

![在庫コントロール画面上部_R.png](https://www.temairazu.com/net/manual/img/在庫コントロール画面上部_R.png)

メッセージ機能を利用するに設定後、手間いらず画面上部に上記アイコンが表示されます。
未読のメッセージがある場合はアイコンに数字が表示されます。

・在庫コントロール画面からアイコンをクリック後：未読のメッセージがある場合

![チャット可能予約一覧1_R.png](https://www.temairazu.com/net/manual/img/チャット可能予約一覧1_R.png)

①全ての未読メッセージが表示され、アイコンの隣に緑色の点が表示されます。
アイコンをクリックするとチャット画面が表示されます。

・在庫コントロール画面からアイコンをクリック後：未読のメッセージが無い場合

![チャット可能予約一覧_R.png](https://www.temairazu.com/net/manual/img/チャット可能予約一覧_R.png)

検索画面が表示されます。条件を入力し検索することで、条件に該当する予約が表示されます。

メッセージ項目について
全て：未読・既読にかかわらず、最新のメッセージがある予約を一番上に表示します。
未読：未読メッセージが1件以上ある予約のみを表示します。
既読：全てのメッセージが既読の予約のみを表示します。 

【チャット画面】

![チャット可能詳細画面1_R.png](https://www.temairazu.com/net/manual/img/チャット可能詳細画面1_R.png)

①ゲスト名(宿泊者氏名)が表示されます。
②ゲストが送信したメッセージが表示されます。
③施設が送信したメッセージが表示されます。

【注意事項】
①ゲストにメッセージを送信可能な期間について
〈Booking.com〉
施設は以下の期間でゲストへメッセージを送信することが可能です。
・予約がキャンセルされた場合：予約時から予約キャンセル後7日間まで
・予約がキャンセルされていない場合：予約時からゲストのチェックアウト後7日間まで
ゲストが予約に関してメッセージを送った場合、キャンセル日やチェックアウト日に関わらず
施設は追加で14日間返信可能です。
現在、ゲストは予約時からチェックアウト後66日間、施設にメッセージを送信することが可能です。

〈Airbnb〉
施設は以下の期間でゲストへメッセージを送信することが可能です。
・予約がキャンセルされた場合：予約時から予約キャンセル後20日間まで
・予約がキャンセルされていない場合：予約時からゲストのチェックアウト後20日間まで

〈Trip.com（New）〉
施設は以下の期間でゲストへメッセージを送信することが可能です。
・予約がキャンセルされた場合：予約時から予約キャンセル後3日、
またはゲストの最終メッセージ後3日のいずれか遅い方まで
・予約がキャンセルされていない場合：予約時からゲストのチェックアウト後3日、
またはゲストの最終メッセージ後3日のいずれか遅い方まで

②送信済みのメッセージ編集・取り消しについて
〈Booking.com・Trip.com（New）〉
送信済みのメッセージを編集や取り消しをすることができません。

〈Airbnb〉
Airbnbの管理画面上では送信後15分以内であればメッセージを編集することが可能ですが、
TEMAIRAZU画面から送信済みのメッセージを編集することはできません。
※ゲスト側でメッセージが編集された場合や施設の管理画面からメッセージが編集された場合は、
TEMAIRAZU画面でも編集後の内容に更新されます。
Airbnbの管理画面上では送信後24時間以内であれば送信を取り消すことが可能ですが、
TEMAIRAZU画面から送信済みのメッセージを取り消しすることはできません。

③画像の添付について
1メッセージにつき、1枚の画像添付が可能です。
複数の画像を送信したい場合は、メッセージを分けてそれぞれに画像を添付することで送信が可能です。

※メッセージは1,000件まで保存されます。1,000件を超える場合、古いものから削除されます。
