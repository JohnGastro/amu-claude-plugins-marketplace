# 期間指定集計

> id=218 ／ order=256 ／ 出典: 手間いらず Webマニュアル

期間指定集計

【オプション】＞【ブッキングアナライザー】＞【期間指定集計】

サイト毎や部屋毎の予約データを、期間を指定して集計・確認できます。

![【BA】期間指定集計（サイト別）_R.png](https://www.temairazu.com/net/manual/img/【BA】期間指定集計（サイト別）_R.png)

①集計方法を選択します。

②集計項目を選択します。

③集計対象期間を選択します。

④サイトや部屋で絞り込みができます。

⑤集計結果をグラフで表示します。

⑥グラフの凡例です。

⑦集計データをCSV形式またはPDF形式でダウンロードします。
