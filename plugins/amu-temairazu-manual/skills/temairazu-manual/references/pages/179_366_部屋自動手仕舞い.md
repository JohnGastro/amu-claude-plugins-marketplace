# 部屋自動手仕舞い

> id=366 ／ order=179 ／ 出典: 手間いらず Webマニュアル

部屋自動手仕舞い

対応バージョン：　手間いらず.ＮＥＴ2
　　　　　　　　　　 　TEMAIRAZU YIELD
　　　　　　　　　　 　手間いらず自動

【ツール】＞【各種ツール】＞【部屋自動手仕舞い】

設定した日時に自動的に部屋の手仕舞いを行うことができる機能です。

![新部屋自動手仕舞い_R.png](https://www.temairazu.com/net/manual/img/新部屋自動手仕舞い_R.png)

①設定をON・OFF切り替えることができます。
　※②の手順で「設定する」をクリック後、切り替え可能です。

②手仕舞い実行日時を設定することができます。（最大365日前まで）
　※ 【料金在庫管理期間】で設定されている期間のみ対象となります。
　※初回設定時のみ「設定する」をクリックすると、現在の状態がONに切り替わります。
　※自動手仕舞いの実行時には以下の設定は自動的に削除・上書きされます。
　　(割引割増設定と人数別販売停止はそのまま残ります。)
　　・部屋詳細設定の予約サイト空室提供設定
　　・詳細料金設定
