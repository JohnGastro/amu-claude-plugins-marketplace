# 当日部屋プラン料金一覧

> id=346 ／ order=117 ／ 出典: 手間いらず Webマニュアル

当日部屋プラン料金一覧

【ツール】＞【料金一覧】＞【当日部屋プラン料金一覧】

日毎にサイトと部屋タイプを絞って、プランの料金を一覧で確認することができます。

![当日部屋プラン料金一覧_R.png](https://www.temairazu.com/net/manual/img/当日部屋プラン料金一覧_R.png)

![当日部屋プラン料金一覧結果_R.png](https://www.temairazu.com/net/manual/img/当日部屋プラン料金一覧結果_R.png)

①表示させたい日程を選択してください。

②予約サイトの部屋名で絞り込みをしたい場合は入力してください。

③プラン名で絞り込みをしたい場合は入力してください。

④料金区分を選択してください。

⑤予約サイトを選択してください。

⑥部屋マスタを選択してください。

⑦「手仕舞い・休止中の部屋プランを除く」場合にはチェックをいれてください。

⑧「検索」クリックします。

⑨①～⑦の条件での結果が下部に表示されます。

⑩マウスオーバーする事で、表示させている日程に設定されている料金ランク名を確認できます。

⑪条件を変更して再度検索したい場合にクリックします。
