# 複数料金グループのカレンダー一括設定

複数料金グループのカレンダー一括設定

対応バージョン：　手間いらず.ＮＥＴ2
　　　　　　　　　　 　TEMAIRAZU YIELD
　　　　　　　　　　 　手間いらず自動

【設定】＞【料金グループ設定】＞任意の料金グループの「カレンダー」ボタン

選択した料金グループのカレンダーをコピーし、複数の料金グループに一括登録することができます。

![料金グループカレンダー一括設定.png](https://www.temairazu.com/net/manual/img/料金グループカレンダー一括設定.png)

①右側の料金グループ一覧からカレンダー登録したい料金グループ名をクリックします。
（ドラッグまたはShiftキー、Ctrlキーを押しながらクリックで複数選択が可能です）
②「<<」をクリックします。
③左側の登録先料金グループに①で選択した料金グループ名が表示されているかを確認してください。
④「登録」ボタンをクリックすると内容確認のポップアップが表示され、「OK」をクリックすると設定が反映されます。
※詳細料金設定をしているプランが含まれる場合、料金グループ別カレンダーをご設定いただくと
　詳細料金設定は無効となりますのでご注意ください。

料金グループカレンダー設定についてはこちらをご覧ください。
