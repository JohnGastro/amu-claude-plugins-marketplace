# プラングループ販売料金再計算

> id=491 ／ order=89 ／ 出典: 手間いらず Webマニュアル

プラングループ販売料金再計算

【料金コントロール】＞【プラングループ一覧】＞画面最下部の関連するページより【プラングループ販売料金再計算】

本画面は、プラングループ設定（詳細）の「料金調整」から料金を直接入力して変更している状態を
一括で元のプラングループに設定した料金設定に戻したい（再計算したい）場合に利用できます。

この際、「料金調整」を行っていないプラングループも基本料金カレンダーの設定に準じた料金設定に戻ります。
また、デライブドプライシングの利用設定を変更した際、既に設定済みの料金を適用する場合に利用できます。
プラングループ詳細設定、またはプラングループカレンダーを設定している場合、設定を加味した料金に再計算します。

![プラングループ販売料金再計算_R.png](https://www.temairazu.com/net/manual/img/プラングループ販売料金再計算_R.png)

①再計算したいプラングループにチェックを入れてください。

②「再計算」をクリックしてください。

③再計算した内容を各予約サイトへ反映させる為に必ずプラン料金のリフレッシュ更新を実行してください。
