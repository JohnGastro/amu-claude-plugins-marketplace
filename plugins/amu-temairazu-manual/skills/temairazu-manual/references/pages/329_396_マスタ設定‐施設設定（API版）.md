【設定】＞【予約サイト設定】＞【管理画面】（手間なしNEXT）＞【マスタ設定】＞【施設設定】
【オプション】＞【手間なしNEXT】＞【マスタ設定】＞【施設設定】
※本項目は、API版をご利用の施設様が対象です。

![手間なしNEXT施設設定API版102_R.png](https://www.temairazu.com/net/manual/img/手間なしNEXT施設設定API版102_R.png)

①一元在庫サービス - 施設設定
　 一元在庫サービス用の施設情報を設定することができます。
　 設定方法についてはこちら
　 ※「手間なしNEXT」で販売を開始するには、「一元在庫サービス - 施設設定」と「手間なしNEXT - 施設設定」の両方が
　 設定されている必要があります。

②手間なしNEXT施設番号
　自動で付番されます。予約ページ制作ご担当者様にご提出ください。

③予約確認画面URL
　ご提供いただいたURLが表示されます。

④ヘッダー、フッター、コピーライト、利用規約、プライバシーポリシー
　設定は不要です。

⑤トラッキングID
　Googleの計測タグをご利用の場合、トラッキングIDを設定してください。

⑥タグマネージャー
　タグマネージャーのタグを設置する場合、
　「タグマネージャーの利用規約を確認する」をクリックして規約に同意してください。
　同意後にコンテナIDを設定してください。
　※事前にタグマネージャーの画面でコンテナの設定をお願いいたします。

⑦利用カード決済サービス
　利用したい決済システムを選択してください。
　※API版のご利用で事前決済の設定を行う場合、GMOまたはJTB Book&Payのご利用が必須となります。
　※各決済システム会社とのご契約お手続きにつきましては、弊社担当営業にお問い合わせください。
　※決済システム会社をご利用でない場合は「なし」を選択してください。（現地決済になります）
　GMOの設定はこちらをご確認ください。

⑧メールアドレス
　予約通知を受け取るメールアドレスを設定してください。カンマ区切りで5つまで登録できます。

⑨緯度/経度
　緯度/経度の設定ができます。緯度/経度の値はGoogleマップ等でご確認ください。

⑩子供設定
　設定する子供区分の「利用する」にチェックを入れて
　ユーザー画面に表示される子供の名称を入力してください。
　一元在庫サービス＞プラン設定＞子供料金設定と紐付きます。 
　＜例＞
　子供A区分の「利用する」にチェックを入れて子供の名称を「小学生」と設定した場合
　一元在庫サービス＞プラン設定＞子供料金設定の子供Aと「小学生」が紐付きます。

⑪子供区分優先順位設定
　プランオプション設定の子供追加料金設定において、無料対象となる子供区分の優先順位を設定できます。

⑫検索条件の最大値/最小値
　予約ページでユーザーが指定する検索値の上限値と下限値の設定です。
　施設様の受け入れ可能な値を設定してください。

⑬他社料金比較設定
　ユーザー画面の空室カレンダーで空室がある日付を選択し、
　「検索する」をクリックした後に「他社サイト料金と比較する」をクリックすると、
　チェックを入れた他社サイトの料金が表示されるようになります。

　＜設定手順＞
　1. 「予約サイト一覧」から、利用する予約サイトの「モジュール利用」が「ON」になっていることをご確認ください。
　2. 「アカウント設定」から、利用する予約サイトの施設情報が設定されていることをご確認ください。
　3. 利用する予約サイトの他社料金比較設定を有効にしてください。
　4. 部屋タイプ紐付けのリンクをクリックして、各サイトの部屋と部屋タイプの紐付けをしてください。
　　

![他社料金比較部屋タイプ紐付け.png](https://www.temairazu.com/net/manual/img/他社料金比較部屋タイプ紐付け.png)

　　4-1.キーワードを指定し表示を絞込みできます。
　　4-2.部屋タイプの紐付けを行ってください。
　　　　　※部屋プラン一覧で設定している部屋の紐付けと完全に一致させていただく必要がございます。
　　4-3.「登録する」をクリックしてください。 
　　4-4.「各種設定の最新化」をクリックしてください。

⑭Google Free Booking Link設定
　Google Free Booking Linkのランディング画面から施設様の予約画面へ遷移するためのURLを設定してください。
　※手間いらず.NET2またはTEMAIRAZU YIELDをご利用の施設様のみ設定可能です。

⑮「登録・更新」をクリックしてください。

設定後は必ず手間なしNEXT管理画面右上の「各種設定の最新化」をクリックしてください。

⑯特定予約制限設定画面に遷移します。※事前に設定した宿泊者からの予約をブロックする機能です。

予約制限条件設定

![予約制限20250529_01.png](https://www.temairazu.com/net/manual/img/予約制限20250529_01.png)

①予約制限の条件を追加登録できます。

![予約制限20250529_02.png](https://www.temairazu.com/net/manual/img/予約制限20250529_02.png)

②CSVファイルを用いて予約制限の条件を一括登録できます。

![予約制限20250529_03.png](https://www.temairazu.com/net/manual/img/予約制限20250529_03.png)

　②-1　一括登録用のフォーマット（CSVファイル）をダウンロードできます。
　②-2　編集後のCSVファイルを選択します。
　②-3　編集後のCSVファイルをアップロードし、予約制限条件を一括登録できます。

　※条件を入力する際の注意点につきましては、ダウンロードボタンの上の「入力形式」を必ずご確認ください。
　※CSVファイルのみ対応しております。アップロードが正常に完了しない場合、ファイルの保存形式をご確認ください。
　※新規登録にのみ対応しております。

③登録済みの予約制限条件が表示されます。

④登録済みの予約制限条件を編集できます。

⑤ゴミ箱アイコンをクリックすることで、登録済みの予約制限条件の削除ができます。
　チェックボックスと「一括削除」ボタンを用いて一括削除も可能です。

設定後は必ず手間なしNEXT管理画面右上の「各種設定の最新化」をクリックしてください。
