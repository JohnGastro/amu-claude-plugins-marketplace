# プラン検索詳細設定

プラン検索詳細設定

【設定】＞【プラン検索詳細設定】

日付やプラン名などを指定して該当するプランを抽出し、個別、または一括で【詳細料金設定】を行います。
※手仕舞い解除の設定をする場合は【提供しない期間・曜日の設定】で手仕舞いにしている設定を
　上書きしてしまいますので十分ご確認ください。

![プラン検索詳細設定新_R.png](https://www.temairazu.com/net/manual/img/プラン検索詳細設定新_R.png)

①ここで入力した日付に売り出されているプランについて検索します。
　※料金未設定、読取専用、料金更新非対応のプラン、または入力した日付が
　　 プランの販売期間外の場合は検索結果に表示されませんのでご注意ください。

②プラン名で検索します。プラン名もしくはプラン名の一部を入力してください。
　 また、除外キーワードを設定することでさらに絞り込むことができます。

③タグで絞り込みを行います。

④予約サイトで絞り込みを行います。
　※いずれにもチェックがされていない場合は全てのサイトが対象となります。

⑤紐づいている部屋で絞り込みを行います。

⑥プラン料金で絞り込みを行います。
　※設定されたランク料金の最小受け入れ区分の料金のみ対象となります。

⑦検索対象に休止中のプランを含めるかを選択できます。

⑧検索対象のプラン手仕舞いの有無などを選択できます。

⑨検索条件を入力した後、検索をクリックしてください。
　※設定を行うには検索結果件数を400件以内に収めてください。

![プラン検索詳細設定2キャプチャ_R.PNG](https://www.temairazu.com/net/manual/img/プラン検索詳細設定2キャプチャ_R.PNG)

【A】一括で設定を行う場合

　検索されたプランでチェックが入っているものについて一括で設定する場合は、
　「一括で設定する」をクリックし、「OK」をクリック。　出てきた画面で設定を行ってください。
　※一度に設定できる期間は31日間以内までとなります。
　※「設定を解除する」をクリックした場合は設定した期間内の【詳細料金設定】を全て解除します。

【B】個別に設定を行う場合

　検索されたプランでチェックが入っているものに関して個別に設定を行いたい場合は、
　個別に設定条件を設定し、上部の「個別設定を登録する」をクリックしてください。
　※一度に設定できる期間は31日間以内までとなります。
　※「設定を解除する」をクリックした場合は設定した期間内の【詳細料金設定】を全て解除します。
