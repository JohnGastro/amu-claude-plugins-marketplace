# ダッシュボード

> id=528 ／ order=317 ／ 出典: 手間いらず Webマニュアル

ダッシュボード

【設定】＞【予約サイト設定】＞【管理画面】（手間なしNEXT）＞【ダッシュボード】
【オプション】＞【手間なしNEXT】＞【ダッシュボード】

デフォルトでは、すべての項目が閉じた状態で表示されます。
各項目左側のピン留めアイコン「

![ピン止め.png](https://www.temairazu.com/net/manual/img/ピン止め.png)

」「

![ピン止め解除.png](https://www.temairazu.com/net/manual/img/ピン止め解除.png)

」をクリックすることで、
項目を開いたまま、または閉じたままの状態に固定することができます。

＜登録情報＞
予約トップ画面や販売先限定のURL、
自動で付番される手間なしNEXT施設番号、ご利用中のバージョンを確認できます。

＜初期設定フロー＞
初期設定フローに沿って各項目を設定していただくことで、手間なしNEXTでの販売を開始できます。
※赤背景色で表示されている「設定必須」項目のみの設定でもご利用いただけます。
詳細は、概要・初期設定フローをご確認ください。

＜予約分析＞

![グラフ表示設定01_01_R.png](https://www.temairazu.com/net/manual/img/グラフ表示設定01_01_R.png)

①-1 「宿泊数と宿泊売上」はチェックイン日を基準に集計されます。
①-2 「予約数と予約売上」は予約日を基準に集計されます。
①-3 「リードタイム集計」はどれくらい前に予約が成立しているのかを集計することができます。
　　　 年月指定・集計種別・単位を選択できます。
　　　 集計単位を「％」にした場合は下記の計算式になります。

　　　 〈計算式〉
　　　　(年月指定で設定した年月に成立した予約のうち、リードタイムの条件を満たす予約数(予約室数)) / (年月指定で
　　　　 設定した年月の全予約数(予約室数))×100

　　　 〈設定内容例：チェックイン日から3週間前の予約室数の割合〉
　　　　年月指定：2026/01
　　　　種別：予約室数
　　　　単位：％
　　　　・2026年1月に成立した全予約室数が40室
　　　　・2026年1月に成立した予約のうち、チェックイン日から3週間前の予約室数10室

　　　　〈設定内容例の計算式〉
　　　　10室/40室×100=25％

②予約データを集計し、グラフ表示する際の集計期間単位（例：日別・週別・月別など）を選択できます。
③集計の基準となる日付を選択できます。
④表やグラフに一度に表示する期間を選択できます。
　 ※②の集計期間単位により、選択できる期間が異なります。
　 ・日単位：7日、14日、21日、28日から選択
　 ・週単位：4週間、8週間、12週間、16週間、20週間、24週間から選択
　 ・月単位：12か月、24か月、36か月、48か月、60か月、72か月から選択
　 ・年単位：選択項目なし（2021年を下限として、最大6年分まで表示可能です）
⑤別年の同時期と予約状況を比較するかどうかを選択できます。
　 ※②の集計期間単位で「日単位」または「月単位」を選択している場合に利用可能です。
　　「対比する」を選択した場合、2021年を下限として、昨年から最大6年分までさかのぼって比較できます。
　 なお、「月単位」で対比するを選択した場合、表示される月数は12か月です。
⑥表示している内容をPDF、CSVでダウンロードできます。

＜会員分析＞

![手間なしNEXT会員分析.png](https://www.temairazu.com/net/manual/img/手間なしNEXT会員分析.png)

①「今月獲得した会員数」今月中に新規登録された会員数を表示します。
　「累計会員数」現在有効な会員の総数を表示します。
②累計会員数に対する、性別ごとの会員数とその割合を表示します。
③累計会員数に対する、年代ごとの会員数とその割合を表示します。
④累計会員数のうち、会員数が多い上位10都道府県について、
　都道府県別の会員数とその割合を表示します。
※円グラフ内の文字色は、背景色が濃い場合は白字、薄い場合は黒字で表示されます。

＜部屋・プラン販売状況＞

![部屋・プラン販売状況サマリ.png](https://www.temairazu.com/net/manual/img/部屋・プラン販売状況サマリ.png)

①「一元在庫サービス - 部屋設定」で設定されている部屋数、
　およびその中で「販売中」「準備中」の部屋数、さらに「準備中」の部屋名を確認できます。
②「一元在庫サービス - プラン設定」で設定されているプランのうち、
　販売先に「手間なしNEXT」が含まれているプランの数を表示します。
　また、その中で「販売中」「休止中」のプラン数、ならびに「休止中」のプラン名、
　販売期間が終了しているプラン名、部屋に紐付いていないプラン名も確認できます。

＜販売状況チェックカレンダー＞

![手間なし販売状況チェックカレンダー表示.png](https://www.temairazu.com/net/manual/img/手間なし販売状況チェックカレンダー表示.png)

①販売状況を確認したい日付を選択します。
②表の表示日数を選択します。7・14・21・28日表示から選択可能です。
③販売状況を確認したい部屋を選択します。
④販売状況を確認したいプランを選択します。
⑤上記で選択した条件で表示されます。
⑥〇×で販売状況が分かる一覧が表示されます。〇×部分をクリックで販売状況の詳細確認ができます。

＜手間なしNEXT 月間利用料確認＞

![月間利用料サマリ.png](https://www.temairazu.com/net/manual/img/月間利用料サマリ.png)

月間利用料を確認できます。
過去の利用料を確認する場合は、利用料確認をご参照ください。
