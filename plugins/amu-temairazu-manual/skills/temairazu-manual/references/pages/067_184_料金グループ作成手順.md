# 料金グループ作成手順

> id=184 ／ order=67 ／ 出典: 手間いらず Webマニュアル

料金グループ作成手順

![新UI_料金グループ作成00.png](https://www.temairazu.com/net/manual/img/新UI_料金グループ作成00.png)

料金グループ設定画面で「追加」をクリックします。

![料金グループ紐付け20240328.png](https://www.temairazu.com/net/manual/img/料金グループ紐付け20240328.png)

①料金グループ名を入力します。

②料金グループを最下段、最上段のどちらに追加するか選択します。

③ ④以下の工程で選択したプランが表示されます。選択したプランは「×」で除外できます。

④部屋を選択すると、その部屋に紐付く各サイトのプランが表示されます。プラン名称、サイト選択で表示されるプランを
　 絞り込むことができます。作成する料金グループに紐付けたいプランを「＋」で選択してください。

⑤現在料金グループに紐付けがされていないプランのみ表示したい場合はチェックを入れてください。

⑥登録せずに戻ります。

⑦クリックで登録します。
