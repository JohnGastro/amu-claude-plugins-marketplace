# 部屋グループ一覧

> id=545 ／ order=198 ／ 出典: 手間いらず Webマニュアル

部屋グループ一覧

【ツール】＞【各種ツール】＞【部屋グループ一覧】

ターゲットプライス（手間いらず自動専用）では、設定単位が部屋タイプの場合、
複数の部屋タイプの在庫数や稼働率を条件としたターゲットプライス設定が可能です。
複数の部屋タイプを条件に設定する場合は、事前に本画面で部屋のグループを作成します。

![部屋グループ一覧01.png](https://www.temairazu.com/net/manual/img/部屋グループ一覧01.png)

①作成済みの部屋グループを検索できます。

②新しい部屋グループを作成できます。

③作成済みの部屋グループが一覧表示されます。
　部屋グループ名をクリックすると、名称の変更および紐付ける部屋マスタの変更が可能です。

④不要な部屋グループを1件ずつ、または一括で削除できます。

![部屋グループ一覧02.png](https://www.temairazu.com/net/manual/img/部屋グループ一覧02.png)

②-1 部屋グループ名を入力します。

②-2 部屋グループに紐付ける部屋マスタを選択します。

※部屋グループを条件としたターゲットプライス設定方法については、詳細設定をご参照ください。
