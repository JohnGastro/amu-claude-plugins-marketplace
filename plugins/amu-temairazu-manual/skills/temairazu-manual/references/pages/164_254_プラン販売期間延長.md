# プラン販売期間延長

プラン販売期間延長

対応バージョン：　手間いらず.ＮＥＴ2
　　　　　　　　　　　TEMAIRAZU YIELD
　　　　　　　　　　　手間いらず自動

対象予約サイト：
・楽天トラベル・じゃらん・一休.com・ベネフィットステーション・予約番V4・ゆこゆこ・ホテリブ／セキララ
・タビチャットエンジン・Ohmyhotel&co・スカイマーク たす旅・RC Booking・予約プロクロス・一元在庫サービス

【ツール】＞【各種ツール】＞【プラン販売期間延長】

複数の予約サイトのプランの販売期間を一括で延長することができます。

![プラン販売期間延長プラン毎に管理_R.png](https://www.temairazu.com/net/manual/img/プラン販売期間延長プラン毎に管理_R.png)

①「プラン毎に管理」をクリックしてください。

![プラン販売期間延長_R.png](https://www.temairazu.com/net/manual/img/プラン販売期間延長_R.png)

②表示するプランを絞り込むことができます。

③販売期間を延長するプランをチェックしてください。

④販売期間終了日を入力してください。

⑤「登録」をクリックすると設定が反映されます。

※一元在庫サービスについては販売期間終了日と同日で掲載期間終了日も更新されます。
