# 料金ランク自動作成機能

> id=255 ／ order=61 ／ 出典: 手間いらず Webマニュアル

料金ランク自動作成機能

【ツール】＞【各種ツール】＞【料金ランク自動作成】

既存の料金ランクをベースに、新たな料金ランクを自動作成する機能です。

例：「年末年始」という名前のランクを設定しています。
　　素泊まりプランは 「年末年始」ランクを15,000円、朝食プランは 「年末年始」ランクを18,000円で販売しており、
　　これらから一律で1000円割引した「お盆」ランクを作りたいとします。

　　通常であれば、料金ランク管理から「お盆」ランクを追加し、プラン毎の料金設定や料金グループ設定で
　　料金設定をする必要がありますが、 一律で割引割増したランクを作成する場合は、本機能からランクの追加と
　　料金設定を同時に行うことができます。

![新料金ランク自動作成機能_R.png](https://www.temairazu.com/net/manual/img/新料金ランク自動作成機能_R.png)

①どのランクを基準にして新規ランクを作成するか選びます。

②割引・割増率を設定します。

③新規追加するランク名とランクの色を設定します。

④既存のすべてのランクをチェックして一括で割引・割増率を設定できます。

⑤割引・割増率適用時に端数が出る金額の処理方法を指定できます。

⑥「ランク作成」をクリックすると新規ランクが作成されます。

※注意

・本機能は料金が設定されているすべての「プラン料金」と「グループ料金」が対象となります。
　一部プラン、グループのみ除外することはできません。

・既存の料金ランクの料金が0円に設定されている区分は、新しい料金ランクでも0円で登録されます。
　0円ランクに1000円割増をしても新規ランクは0円で登録されます。

・作成された料金ランクは自動計算が完了する前に料金が入力されないよう
　プラン毎、また料金グループの料金設定画面では表示されません。
　自動計算完了後に自動で表示に更新されます。

・本機能を実行すると、処理が完了するまで二回目以降の実行はできません。
　処理中は「在庫コントロール画面」に処理中の旨のメッセージを表示します。
