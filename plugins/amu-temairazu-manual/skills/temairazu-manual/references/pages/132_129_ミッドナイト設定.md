# ミッドナイト設定

> id=129 ／ order=132 ／ 出典: 手間いらず Webマニュアル

ミッドナイト設定

対応バージョン： 手間いらず.ＮＥＴ2
　　　　　　　　　　　TEMAIRAZU STANDARD
　　　　　　　　　　　TEMAIRAZU ADVANCE
　　　　　　　　　　　TEMAIRAZU YIELD
　　　　　　　　　　　手間いらず自動

【設定】＞【基本設定】＞【ミッドナイト設定】

各予約サイトで24時以降のミッドナイト販売をされる在庫料金を、設定した時間まで管理できる機能です。

対応宿泊予約サイト
「楽天トラベル」「じゃらん」「一休.com」「ヤドバンス」「OPTIMA（※）」「triplaホテルブッキング」「Trip.com」「Trip.com(New)」「予約プロクロス」

※OPTIMAは、アカウント設定のログインIDが「z」から始まるご施設様のみ本機能をご利用可能です。

・設定方法

![新ミッドナイト設定111.png](https://www.temairazu.com/net/manual/img/新ミッドナイト設定111.png)

①「ミッドナイト設定切り替え時間」を選択
②「登録」をクリック

上記設定を行うと、24時以降「ミッドナイト設定切り替え時間」までは在庫コントロール画面で「今日」を表示しても
前日の日付が先頭に表示され、料金・在庫の更新が可能となります。また、部屋詳細設定、詳細料金設定、
プラン検索詳細設定などもご利用いただけます。

※設定を変更した場合は、該当サイトの部屋プランを再度取得する必要がございます。

「一休.com」「ヤドバンス」「OPTIMA」「triplaホテルブッキング」「Trip.com」「Trip.com(New)」「予約プロクロス」
でミッドナイト設定をご利用される場合は、下記手順で設定をしてください。

（予約サイト管理画面での操作）
1. プランの手仕舞い設定を0日前、24時以降の設定をする。

（手間いらず画面での操作）
2. 設定＞予約サイト設定＞部屋プラン一覧＞「予約サイトから部屋とプランの情報を取得する」を実施する。
3. 設定＞基本設定＞ミッドナイト設定より「ミッドナイト設定切り替え時間」を設定する。
4. 設定＞予約サイト設定＞アカウント設定より「ミッドナイト設定切り替え時間」を設定する。

・「ミッドナイト設定」後の【在庫コントロール画面】

![新ミッドナイト設定2_R.png](https://www.temairazu.com/net/manual/img/新ミッドナイト設定2_R.png)

24時以降、設定した「ミッドナイト設定切り替え時間」までは前日の日付に [深夜] と表示されます。
