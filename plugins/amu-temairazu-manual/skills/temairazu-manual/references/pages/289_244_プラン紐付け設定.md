# プラン紐付け設定

> id=244 ／ order=289 ／ 出典: 手間いらず Webマニュアル

プラン紐付け設定

【各種設定】＞【プラン紐づけ設定】
プラン別予約状況の分析に使用するプランのグループを作成します。
サイト上で作成したプランを各施設のプランと紐付けることができます。

![honbu7.5.1_R.png](https://www.temairazu.com/net/manual/img/honbu7.5.1_R.png)

①紐付けたプランの名称を作成します。

②紐付け先の施設名です。

③紐付けする部屋名を選択します。

④紐付けするプラン名を選択します。

⑤プラングループの編集・削除ができます。

⑥クリックするとプラングループを新規作成します。
