# 日毎（全部屋）の手仕舞い

日毎（全部屋）の手仕舞い

【在庫コントロール画面】

![roomdetail_tejimai3-2-1_R.png](https://www.temairazu.com/net/manual/img/roomdetail_tejimai3-2-1_R.png)

①日毎の「全部屋詳細設定ボタン」をクリック。

![roomdetail_tejimai3-2-2_R.png](https://www.temairazu.com/net/manual/img/roomdetail_tejimai3-2-2_R.png)

②「全予約サイト一括手仕舞い」にチェックを入れてください。
　※「予約サイト空室提供」ではチェックの有無により空室提供する予約サイトを絞ることができます。
　　予約サイトのプランに詳細料金設定を設定している場合、その設定は解除されますのでご注意ください。

③「登録する」をクリックしてください。

![roomdetail_tejimai777_R.png](https://www.temairazu.com/net/manual/img/roomdetail_tejimai777_R.png)

④設定した日にちには、「手仕舞い」と表示されます。

・緊急手仕舞い

緊急で手仕舞いを行いたいとき、優先的に各予約サイトへ手仕舞い情報を反映させることができます。

![roomdetail_tejimai555_R.png](https://www.temairazu.com/net/manual/img/roomdetail_tejimai555_R.png)

①【設定】＞【基本設定】＞【在庫コントロール画面設定】より、
　 緊急手仕舞いを「使用する」に選択してください。

②「設定する」をクリックしてください。
　在庫コントロール画面に緊急手仕舞いのON/OFFを切り替えるボタンが表示されます。

![roomdetail_tejimai666_R.png](https://www.temairazu.com/net/manual/img/roomdetail_tejimai666_R.png)

③「OFF」をクリックすると、全部屋に手仕舞いがかかり表示が「ON」になります。
　 また「ON」をクリックすると、手仕舞いが解除され、表示が「OFF」になります。
