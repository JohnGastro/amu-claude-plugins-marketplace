# 部屋をサイトに紐付ける

> id=274 ／ order=169 ／ 出典: 手間いらず Webマニュアル

部屋をサイトに紐付ける

手間いらずで作成した部屋のデータと連携先サイトの部屋を紐付けることができます。

【ツール】＞【各種ツール】＞【プラン一括管理】＞【部屋一覧】

![【プラン一括】紐づけ1_R.png](https://www.temairazu.com/net/manual/img/【プラン一括】紐づけ1_R.png)

①紐付けをしたい部屋の「部屋紐付設定」をクリックしてください。

![【プラン一括】紐づけ2_R.png](https://www.temairazu.com/net/manual/img/【プラン一括】紐づけ2_R.png)

②手間いらずで作成した部屋名が表示されます。
　ペンアイコン「

![編集アイコン部屋紐付け.png](https://www.temairazu.com/net/manual/img/編集アイコン部屋紐付け.png)

」をクリックすると部屋情報の編集を行えます。

③紐付ける部屋を選択してください。
　※プルダウンで紐付ける部屋が出てこない場合は、
　　「部屋取得」をクリックし、「部屋とプランの情報を取得する」をクリックしてください。
　　詳細は事前の操作をご覧ください。

④「紐付開始」をクリックしてください。
　一括で紐付けを行う場合は「一括紐付」をクリックしてください。

⑤「一括登録」をクリックするとサイト側に部屋を作成または更新することができます。
　 ※対象予約サイト：Booking.com
