# グループサイズ分析

> id=526 ／ order=261 ／ 出典: 手間いらず Webマニュアル

グループサイズ分析

【オプション】＞【ブッキングアナライザー】＞【グループサイズ分析】

各種集計データを、1予約あたりの人数別に確認できます。

![グループサイズ分析1.png](https://www.temairazu.com/net/manual/img/グループサイズ分析1.png)

①集計方法を選択します。

②「平日」「休前日」「平日・休前日」のいずれかを選択します。
　平日･･･日～木曜日（休前日に該当しない日）
　休前日･･･金～土曜日、祝前日

③集計対象月を選択します。

④部屋で絞り込みができます。

⑤集計データをCSV形式またはPDF形式でダウンロードします。

⑥合計人数が表示されます。
　※子供人数や男女人数の扱いは予約サイトによって異なるため、
　　合計人数と各数値の合計が一致しない場合があります。

⑦合計室数が表示されます。

⑧一室一泊あたりの売上が表示されます。

⑨一人一泊あたりの売上が表示されます。

⑩該当するグループサイズの割合が表示されます。

⑪ ⑥～⑩の内容が棒グラフで表示されます。
