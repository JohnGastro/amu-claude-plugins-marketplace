# プラン毎の手仕舞い

> id=45 ／ order=49 ／ 出典: 手間いらず Webマニュアル

プラン毎の手仕舞い

【在庫コントロール画面】

・方法【A】

![新プラン毎手仕舞い3_R.png](https://www.temairazu.com/net/manual/img/新プラン毎手仕舞い3_R.png)

①部屋毎の「部屋詳細設定」をクリックしてください。

![plan_tejimai3-2-2.png](https://www.temairazu.com/net/manual/img/plan_tejimai3-2-2.png)

②部屋詳細設定画面左上の「詳細料金設定をする」をクリックしてください。

![plan_tejimai3-2-3.png](https://www.temairazu.com/net/manual/img/plan_tejimai3-2-3.png)

③予約サイト名横の＋をクリック

④プラン毎の手仕舞い項目にチェック

⑤「登録する」をクリックしてください。

![新プラン毎手仕舞い4_R.png](https://www.temairazu.com/net/manual/img/新プラン毎手仕舞い4_R.png)

⑥設定した日付に「詳細設定」と表示されます。

・方法【B】

プラン検索詳細設定にて手仕舞いを設定できます。
設定方法は、こちらを参照してください。
