# 案件登録

> id=259 ／ order=102 ／ 出典: 手間いらず Webマニュアル

案件登録

ここでは団体宿泊予約の案件登録と部屋の割当について説明します。

◆案件登録

【ツール】＞【DANTAI Travel】＞【登録】

![DANTAI案件登録_R.png](https://www.temairazu.com/net/manual/img/DANTAI案件登録_R.png)

「*」の項目は必須入力です。

①予約の段階や状況に応じて、ステータスを選択します。

②案件の管理番号を割り振ります。指定がなければ自動で採番することができます。

③案件タイプの「受注/募集」を選択します。

④団体種別を選択します。
　 (例. 高校の修学旅行→教育旅行）

⑤団体名を入力します。

⑥販売先を選択します。
　 追加で担当者などの情報を登録できます。

⑦催行が未確定の場合、確定期限を入力します。

⑧オプションの予約情報入力機能を利用している場合、案件登録後、部屋割当ができます。

⑨宿泊日を選択します。

⑩確度を選択できます。
