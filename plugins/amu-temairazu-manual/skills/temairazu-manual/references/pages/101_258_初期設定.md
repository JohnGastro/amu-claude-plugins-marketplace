# 初期設定

> id=258 ／ order=101 ／ 出典: 手間いらず Webマニュアル

初期設定

ここではDANTAI Travelを初めてご利用いただく際の初期設定の方法をご説明します。

◆販売先の設定（必須）

【ツール】＞【DANTAI Travel】＞【販売先】

団体宿泊予約の販売先（旅行会社等）の情報を登録します。

![DANTAI販売先1_R.png](https://www.temairazu.com/net/manual/img/DANTAI販売先1_R.png)

①「追加」をクリックします。

②ウインドウ表示されるので販売先の名称を入力します。

③「登録」をクリックします。

![DANTAI販売先2_R.png](https://www.temairazu.com/net/manual/img/DANTAI販売先2_R.png)

④販売先に紐づく支店や営業所を登録したい場合に「取扱箇所設定」をクリックします。　※任意の設定です。

⑤名称等の情報を入力します。

⑥「登録」をクリックします。

![DANTAI販売先3_R.png](https://www.temairazu.com/net/manual/img/DANTAI販売先3_R.png)

⑦登録した販売先情報は上図のよう表示されます。

⑧登録情報を削除したい場合はゴミ箱アイコンをクリックします。

⑨販売先/取扱箇所をCSVダウンロードできます。

⑩CSVダウンロードしたデータをアップロードできます。
※アップロードした場合、追加のみ可能です。（変更・削除はできません）

⑪登録した販売先を並べ替えすることができます。

◆確度の設定(任意）

【ツール】＞【DANTAI Travel】＞【確度】

「確度」とは催行の可能性がどの程度確実かの度合いを言います。
確度を現す際にどのような名称を表示するかをこの画面で設定します。

![DANTAI確度_R.png](https://www.temairazu.com/net/manual/img/DANTAI確度_R.png)

①確度名称を入力します。
　 ※画面上ではA, B, C, Dとしておりますが、特に指定はありません。

②「更新」をクリックして設定を保存します。
