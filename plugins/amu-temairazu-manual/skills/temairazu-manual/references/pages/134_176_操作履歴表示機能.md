# 操作履歴表示機能

操作履歴表示機能

対応バージョン：　手間いらず.ＮＥＴ2
　　　　　　　　　　　　TEMAIRAZU STANDARD
　　　　　　　　　　　　TEMAIRAZU ADVANCE
　　　　　　　　　　　　TEMAIRAZU YIELD
　　　　　　　　　　　　手間いらず自動

【ツール】＞【各種ツール】＞【施設操作履歴検索】

「操作履歴表示機能」とは、
手間いらず上で行った操作の履歴を確認・把握することができる機能です。

![施設操作02_R.png](https://www.temairazu.com/net/manual/img/施設操作02_R.png)

①操作履歴の取得期間を設定できます。

②検索したいスタッフ（手間いらずアカウント）を選択することができます。

③履歴を検索したい手間いらず上の画面や操作を選択することができます。
※いずれにもチェックがされていない場合は全て対象となります。

④画面/操作に○印が付いている項目に関しては、チェックイン日など更新の対象となる日を指定することができます。
※日付を両方設定している場合は設定された期間での検索、片方のみの場合は「以降」または「以前」での検索となります。
（例：2016-12-14～2016-12-20の取得期間内で、チェックイン日「2017-01-01～2017-01-05」の操作履歴を調べたいときは
「2017-01-01～2017-01-05」を入力します。）

⑤予約サイト名などの任意のキーワードを設定することができます。
キーワードを複数指定したい場合はスペースを間に入れてください。また、その際は「いずれかを含む」を設定してください。

⑥指定した条件で検索します。
