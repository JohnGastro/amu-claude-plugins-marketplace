# 【予約通知印刷機能】 として利用するケース

> id=191 ／ order=222 ／ 出典: 手間いらず Webマニュアル

【予約通知印刷機能】 として利用するケース

【オプション】＞【予約カード印刷設定】

![予約通知印刷機能.png](https://www.temairazu.com/net/manual/img/予約通知印刷機能.png)

「【予約通知印刷機能】として使用する (用紙サイズはA4固定)」にチェックを入れると、
弊社指定の固定フォーマットで印刷することができます。

※用紙サイズはA4固定となります。
