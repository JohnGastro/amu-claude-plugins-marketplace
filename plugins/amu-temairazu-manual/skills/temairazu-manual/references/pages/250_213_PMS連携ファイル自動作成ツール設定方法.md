# PMS連携ファイル自動作成ツール設定方法

> id=213 ／ order=250 ／ 出典: 手間いらず Webマニュアル

PMS連携ファイル自動作成ツール設定方法

![PMSTool1.png](https://www.temairazu.com/net/manual/img/PMSTool1.png)

※[×]をクリックしてPMS連携ファイル自動作成ツールを終了している状態の場合、
　予約データはPMS側に出力されませんのでご注意ください。

①【オプション】＞【PMS項目設定】＞【施設データ情報】に表示されている
　PMS連携用アカウントのID・パスワードを入力してください。

②PMSへの出力形式を選択してください。
　 ※出力形式については、ご利用中のPMSシステム会社様へご確認のうえ設定してください。

③設定した時間間隔ごとに、予約情報を出力します。

④PMSへのデータ出力先のパス（出力先）を指定してください。
　 ※データ出力先については、ご利用中のPMSシステム会社様へご確認のうえ設定してください。

⑤「開始」をクリックすると、PMSへ予約情報が出力されます。
　※「停止」状態の場合、予約情報は出力されません。

⑥自動出力ではなく、手動で出力を行う際に使用します。

＜設定クリック時の画面＞

![PMSTool2.png](https://www.temairazu.com/net/manual/img/PMSTool2.png)

⑦タスクトレイへ格納するかどうかを設定できます。

⑧PC起動時に本ツールを自動で起動する設定ができます。

⑨PMSへ出力するデータを、別の保存先にも同時に出力し、
　バックアップとして保管することができます。

⑩PMS連携ファイル自動作成ツールを、一定時間ごとに再起動する設定ができます。
