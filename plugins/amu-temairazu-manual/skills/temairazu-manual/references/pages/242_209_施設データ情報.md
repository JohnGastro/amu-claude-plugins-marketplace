# 施設データ情報

> id=209 ／ order=242 ／ 出典: 手間いらず Webマニュアル

施設データ情報

【オプション】＞【PMS項目設定】＞【施設データ情報】

![PMS_006.png](https://www.temairazu.com/net/manual/img/PMS_006.png)

①任意設定です。
　ご利用中のPMSシステム会社と完全一致する施設コードを入力してください。

②任意設定です。
　ご利用中のPMSシステム会社と完全一致する施設名を入力してください。

③PMSシステム会社によって仕様が異なります。
　詳細については、ご利用中のPMSシステム会社へお問い合わせください。

④PMSシステム会社へ出力する予約情報の条件を選択します。

⑤ダミーの新規予約情報に続けて、「キャンセル」および「変更」の予約情報を出力します。
　ダミーの新規予約情報は、状態フラグが「新規」となるのみで、
　予約内容は「キャンセル」および「変更」と同一です。
　なお、本機能はXML形式で連携している場合のみ有効です。
　※一度出力された予約情報を、手間いらずの操作で「未出力」に戻した場合は対象外となります。

⑥PMS連携およびPMS項目設定のみ操作可能なアカウントです。
