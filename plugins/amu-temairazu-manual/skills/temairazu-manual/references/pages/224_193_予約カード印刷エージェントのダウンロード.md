# 予約カード印刷エージェントのダウンロード

> id=193 ／ order=224 ／ 出典: 手間いらず Webマニュアル

予約カード印刷エージェントのダウンロード

【オプション】＞【ツールダウンロード】　

予約カード印刷時に必要な「予約カード印刷エージェント」をダウンロードします。

※本ツールの利用には、.NET Framework 3.5 以上が必要です。
　.NET Frameworkがインストールされていない場合は、
　 「.NET Framework3.5」をインターネットで検索し、ダウンロードしてください。
※macOSでは本ツールをご利用いただけません。

お使いのパソコンに.NET Frameworkがインストールされているかは、
以下の手順で確認してください。

■Windows 10 / 11の場合
Windowsキー＞Windowsシステムツール＞コントロールパネル＞
（表示方法が「カテゴリ」の場合）プログラム＞プログラムと機能＞「Windows機能の有効化または無効化」にて、
「.NET Framework 3.5 (.NET 2.0 および 3.0 を含む）」にチェックが入っているか確認してください。

![予約カード印刷ダウンロード_R.png](https://www.temairazu.com/net/manual/img/予約カード印刷ダウンロード_R.png)

※予約カードは、予約カード印刷エージェント起動時のみ印刷されます。
※ツールは、デスクトップへの保存を推奨します。

■ツールダウンロード時のzip形式・exe形式の使い分け
　 施設様のセキュリティ設定により、exe形式のファイルをダウンロードできない場合があります。
　 その場合は、zip形式でダウンロードし、展開または解凍してください。
