# 機能概要

> id=340 ／ order=93 ／ 出典: 手間いらず Webマニュアル

機能概要

予約情報を宿泊証明書としてPDFデータでダウンロードが可能です。

<宿泊証明書の出力操作手順>

【ツール】＞【宿泊証明書出力】
【ツール】＞【各種ツール】＞【宿泊証明書出力】

![宿泊証明書20240308.png](https://www.temairazu.com/net/manual/img/宿泊証明書20240308.png)

①出力したい対象の予約情報を指定し検索ボタンをクリックしてください。

②宿泊証明書の出力言語、プラン名・合計金額の出力有無を設定してください。

③左端にある【出力】ボタンをクリックしてください。

![宿泊証明書出力2.png](https://www.temairazu.com/net/manual/img/宿泊証明書出力2.png)

<宿泊証明書テンプレート出力操作手順>

【ツール】＞【各種ツール】＞【宿泊証明書テンプレート】
宿泊証明書のテンプレートを英語か日本語かを選択し出力できます。
