# 会員管理-会員検索

> id=403 ／ order=321 ／ 出典: 手間いらず Webマニュアル

会員管理-会員検索

【設定】＞【予約サイト設定】＞【管理画面】（手間なしNEXT）＞【会員管理】＞【会員検索】
【オプション】＞【手間なしNEXT】＞【会員管理】＞【会員検索】

![手間なしNEXT会員検索1_03_R.png](https://www.temairazu.com/net/manual/img/手間なしNEXT会員検索1_03_R.png)

①会員氏名や電話番号など条件を指定し、会員情報を検索することができます。
②有効な会員全てをグループ共通会員へ移行します。
※本部管理機能オプションを利用中のご施設様で、本部管理画面の
各種設定＞手間なしNEXTグループ機能＞グループ設定＞施設登録情報一覧
にて「グループ共通会員利用」がONになっている場合に②が表示されます。

![TN会員検索2_02_R.png](https://www.temairazu.com/net/manual/img/TN会員検索2_02_R.png)

③会員検索後に「詳細」をクリックしますと詳細情報が表示されます。

![TN会員検索3_02_R.png](https://www.temairazu.com/net/manual/img/TN会員検索3_02_R.png)

④会員情報の変更と会員を退会させることができます。
