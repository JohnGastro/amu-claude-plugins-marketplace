# 前年比/前々年比

> id=357 ／ order=254 ／ 出典: 手間いらず Webマニュアル

前年比/前々年比

【オプション】＞【ブッキングアナライザー】＞【前年比/前々年比】

指定した期間における前々年・前年・当年・翌年のサイト別予約データを、日毎・月毎で確認できます。
「宿泊人数」で表示した場合、値の下部に（男性人数/女性人数/子供人数）が表示されます。

![【BA】前年比前々年比_R.png](https://www.temairazu.com/net/manual/img/【BA】前年比前々年比_R.png)

①集計方法を選択します。

②集計項目を選択します。

③集計対象日を選択します。

④サイトで絞り込みができます。

⑤集計結果を表で表示します。

⑥集計単位を選択します。

⑦集計データをCSV形式またはPDF形式でダウンロードできます。

⑧集計結果をグラフで表示します。

⑨集計結果のグラフで、対象年の表示／非表示を切り替えることができます。
