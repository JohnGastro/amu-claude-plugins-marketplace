# 部屋タイプ紐付け設定

> id=243 ／ order=288 ／ 出典: 手間いらず Webマニュアル

部屋タイプ紐付け設定

【各種設定】＞【部屋タイプ紐づけ設定】
在庫確認に使用する部屋タイプのグループを作成します。
サイト上で作成した部屋を、複数施設にまたがってグループ化することができます。

![honbu7.4.1_R.png](https://www.temairazu.com/net/manual/img/honbu7.4.1_R.png)

①グルーピングする部屋の名称を作成します。

②紐付け先の施設名です。

③紐付けする部屋名を選択します。

④クリックすると部屋グループを新規作成します。
　 ※設定後は、変更・削除の表記に変わり、部屋タイプグループを編集することができます。
