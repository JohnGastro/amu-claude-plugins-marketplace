# 共通設定

> id=245 ／ order=290 ／ 出典: 手間いらず Webマニュアル

共通設定

【各種設定】＞【共通設定】
本部管理ツール内の表示設定を行うことができます。

![honbu7.6_R.png](https://www.temairazu.com/net/manual/img/honbu7.6_R.png)

①各種分析表において、表示開始日を設定します。
　 デフォルト：当日（月）を中央に表示します。
　 当日（月）を開始日にする：当日（月）を左側に表示します。

②在庫ボックス背景色設定
　 「在庫確認」画面において、設定された在庫数以下になった場合に
　 在庫ボックスの数字の背景色を変更することができます。
