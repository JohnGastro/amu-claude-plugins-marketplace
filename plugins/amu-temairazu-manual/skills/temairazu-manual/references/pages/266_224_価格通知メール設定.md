# 価格通知メール設定

> id=224 ／ order=266 ／ 出典: 手間いらず Webマニュアル

価格通知メール設定

【ツール】＞【各種ツール】＞【レートサーベイ設定】＞【価格通知メール設定】

対象施設の価格通知メールを設定できる機能です。
メール送信設定は、最大10件まで登録できます。

![新UI_レートサーベイ06.png](https://www.temairazu.com/net/manual/img/新UI_レートサーベイ06.png)

①設定の無効/有効を切り替えることができます。
　 チェックを入れた場合のみ、通知メールを受信できます。

②送信するメールの件名を設定できます。

③メールの送信時間を設定できます。

④最安値または最高値のどちらの情報を受信するか選択できます。

⑤通知対象とする部屋を選択できます。

⑥宿泊人数単位または部屋売り料金のいずれで情報を受信するか選択できます。

⑦通知対象とする予約サイトを絞り込むことができます。

⑧設定したキーワードに該当するプランの情報を通知できます。

⑨送信先メールアドレスを最大10件まで設定できます。

⑩設定完了後、「登録」をクリックしてください。

＜価格通知メール例＞
価格通知メール設定を行った際に送信されるメールの例です。

![price4.png](https://www.temairazu.com/net/manual/img/price4.png)

①レートサーベイの対象施設名が表示されます。

②①で表示された施設の施設番号です。

③価格通知メール設定で指定した最安値または最高値が表示されます。

④対象部屋の在庫数が表示されます。

⑤対象の料金・在庫情報が存在しない場合は空白になります。
　 部屋売りのみ対応している施設では、宿泊人数表示を選択すると料金が表示されません。
　対象施設の料金設定が「宿泊人数」か「部屋売り」かをご確認ください。
