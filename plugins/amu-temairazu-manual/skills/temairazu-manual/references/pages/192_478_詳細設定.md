# 詳細設定

> id=478 ／ order=192 ／ 出典: 手間いらず Webマニュアル

詳細設定

【ツール】＞【各種ツール】＞【ターゲットプライス】＞【詳細設定】

![TPJIDO詳細設定1_01_R.png](https://www.temairazu.com/net/manual/img/TPJIDO詳細設定1_01_R.png)

ターゲットプライス設定名をクリックして設定画面を開き、「詳細設定」をクリックします。

※本マニュアルでは、部屋タイプ単位で設定を行う場合を例としてご案内しています。
　料金グループ単位、プラングループ単位で設定する場合は、設定項目が一部異なります。

![TPJIDOの詳細設定の2_02_R.png](https://www.temairazu.com/net/manual/img/TPJIDOの詳細設定の2_02_R.png)

①先に簡易設定を実施することで、詳細設定を容易に作成することができます。

②元となる料金ランクを選択し、料金変動の条件などを設定できます。

![ターゲットプライス2_07_R.png](https://www.temairazu.com/net/manual/img/ターゲットプライス2_07_R.png)

a： 料金変動を適用する部屋タイプを選択します。一度登録すると変更ができません。
　　部屋タイプを変更する場合は、後述の⑫で削除し、設定を作り直してください。

　　ターゲットプライス（手間いらず自動専用）では、設定単位が部屋タイプの場合、
　　部屋グループ一覧で作成済みの部屋グループを選択することで、
　　複数の部屋タイプの在庫／稼働率を条件とした設定が可能です。
　　ターゲットプライス設定で設定単位が「部屋タイプ」、
　　実行条件が「稼働率（部屋毎）」または「残室数」の場合に設定できます。

b： 料金変動を除外する部屋タイプがある場合は選択します。

c： 元となる料金ランクを選択します。

d： 料金変動の設定を行います。
　　※ターゲットプライス設定で選択した実行条件、設定区分により表示される項目は異なります。
　　　dをクリックして選択し、eの料金ランクから設定したいランクをクリックします。

e： 料金ランクが表示されます。
　　下の絞り込み条件を指定することで、表示するランクを絞り込むことができます。

f： dを選択してから「手仕舞い」をクリックすると、手仕舞いを選択できます。
　　手仕舞いを選択中、または設定済みのタブを選択し「手仕舞い設定解除」をクリックすると、
　　手仕舞いを解除した状態になります。
　　※j「登録」ボタン⇒OKを押すまでは未確定です。

g： 「＋」クリックで列・行の追加、「－」クリックで列・行の削除ができます。

h： クリックすると日数・稼働率、および残室数順に並び替えができます。

i： 設定を登録、更新せずに閉じる場合はクリックしてください。

j： 設定を登録、更新して閉じる場合はクリックしてください。

③現在の期間および他の期間からターゲットプライスの詳細設定をコピーできます。

④登録済みの設定が表示されます。
　 部屋タイプ名、料金グループ名、またはプラングループ名をクリックすると編集画面が表示されます。
　 変動履歴の保存設定を有効にしている場合、
　 部屋タイプ名、料金グループ名、またはプラングループ名の先頭にグラフのアイコンが表示されます。

⑤詳細設定を有効にする場合、スイッチをONにしてください。

⑥登録済みの詳細設定をコピーして追加する場合はクリックしてください。

⑦チェックボックスにチェックが入っている詳細設定に対して利用設定を一括で有効化できます。

⑧チェックボックスにチェックが入っている詳細設定に対して利用設定を一括で無効化できます。

⑨チェックボックスにチェックが入っている詳細設定を一括で削除できます。

⑩変動履歴の保存設定ができます。

⑪登録済みの詳細設定を確認できます。

⑫登録済みの詳細設定を1件ずつ削除できます。

■設定単位を「プラングループ」で設定している場合の詳細設定画面

![TPJIDOの詳細設定の3_01_R.png](https://www.temairazu.com/net/manual/img/TPJIDOの詳細設定の3_01_R.png)

デライブドグループ設定を行っているプラングループのためターゲットプライス設定は反映されません。
削除またはコピーのみが操作可能です。
