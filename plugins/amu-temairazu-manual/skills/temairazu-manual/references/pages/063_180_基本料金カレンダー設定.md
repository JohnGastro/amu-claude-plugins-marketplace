# 基本料金カレンダー設定

> id=180 ／ order=63 ／ 出典: 手間いらず Webマニュアル

基本料金カレンダー設定

【設定】＞【料金カレンダー設定】

![基本料金カレンダー設定.png](https://www.temairazu.com/net/manual/img/基本料金カレンダー設定.png)

![カレンダー表示設定.png](https://www.temairazu.com/net/manual/img/カレンダー表示設定.png)

①歯車アイコンクリックでカレンダー表示設定が表示されます。
　「1clickデフォルトランク入力」はクリックするだけで下記で事前設定したランクが自動入力されるボタンです。

①-A 料金ランク名をカレンダー画面で表示させることができます。
①-B 日毎の在庫数をカレンダー画面で表示させることができます。
①-C 1clickデフォルトランク入力の使用有無と使用する場合、平日・休日・休前日でどのランクを設定するか登録できます。

②カレンダー左上のチェックボックスをクリックすると、全日程を選択状態にできます。
　曜日のチェックボックスでその曜日を、週のチェックボックスでその週を全て選択できます。

③料金ランクが表示されます。下の絞り込み条件を指定することで、表示するランクを
　絞り込むことができます。

④詳細料金設定の全プランの料金ランクを上書きする
　詳細料金設定がされているプランのランクを上書きしたい場合はチェックを入れてください。

　部屋詳細設定の全部屋の料金ランクを上書きする
　部屋詳細設定や部屋カレンダーのランクを上書きしたい場合はチェックを入れてください。
　デフォルトは、こちらにチェックが入っています。

⑤選択をリセット･･･カレンダー画面を開き直し、選択を解除します。
　登録･･･カレンダー設定を登録します。1clickデフォルトランク入力をクリックした後も必ずクリックしてください。

⑥当月より先、最大1年間分のカレンダーを一覧表示することができます。
　月毎にExcel形式でのダウンロードも可能です。

カレンダーの設定方法

![新UI_基本カレンダー設定1.png](https://www.temairazu.com/net/manual/img/新UI_基本カレンダー設定1.png)

①ランクを設定したい日をクリックします。選択した日は黄色のマーカーが付きます。
　　※クリックしたままマウスを動かすと、一度に複数の日を選択することができます。
　　※カレンダー左上の全日、曜日、週毎のチェックボックスをクリックすると、
　　　まとめて選択状態にできます。

![新UI_基本カレンダー設定2.png](https://www.temairazu.com/net/manual/img/新UI_基本カレンダー設定2.png)

②右のエリアで料金ランクを選択すると、選択したランクの色がカレンダーに反映されます。

③登録ボタンを押します。確認メッセージの「OK」を押すと、ランクが変更できます。

![新UI_基本カレンダー設定3.png](https://www.temairazu.com/net/manual/img/新UI_基本カレンダー設定3.png)
