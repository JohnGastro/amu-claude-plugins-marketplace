# オリジナルレポート

> id=470 ／ order=214 ／ 出典: 手間いらず Webマニュアル

オリジナルレポート

【ツール】＞【各種ツール】＞【オリジナルレポート】

オリジナルレポートの追加や作成したオリジナルレポートを一覧で確認、変更することができます。

![レポート一覧_R.png](https://www.temairazu.com/net/manual/img/レポート一覧_R.png)

　
①オリジナルレポートを追加することができます。
　※オリジナルレポートの作成方法については、オリジナルレポート設定方法をご参照ください。

②作成したオリジナルレポートの並び順を変更することができます。

③オリジナルレポート名をクリックすると、設定内容の確認および変更ができます。
　作成したレポートをPDFで出力して確認する場合は、レポート名をクリック後、下記のアイコンをクリックしてください。
　　

![レポート一覧②.png](https://www.temairazu.com/net/manual/img/レポート一覧②.png)

④定期報告実行のON／OFFを切り替えることができます。
　ONに設定した場合は、オリジナルレポート設定方法の「報告設定」で設定した内容に基づいて実行されます。

⑤オリジナルレポートの設定内容をコピーし、新たなレポートを作成することができます。

⑥報告設定で設定したメールアドレス宛に、手動でレポートを送信することができます。

⑦オリジナルレポートを、個別または一括で削除することができます。
