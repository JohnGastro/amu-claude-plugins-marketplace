# 機能概要

> id=333 ／ order=94 ／ 出典: 手間いらず Webマニュアル

機能概要

手間いらず画面の右上にある【三（ハンバーガーアイコン）】＞　【デイリーレポート】 

この機能を利用することで、直近2日分の全体または部屋種別毎のチェックイン、チェックアウトの人数・室数や
稼働率の集計結果の確認がリアルタイムで簡単に行えます。PDFダウンロードにも対応してます。

三（ハンバーガーアイコン）場所

在庫コントロール画面、各種設定画面等、手間いらずのどの画面を開いていても右上に表示されます。

![デイリーレポートメニュー.png](https://www.temairazu.com/net/manual/img/デイリーレポートメニュー.png)

①三（ハンバーガーアイコン）にマウスオーバーしてデイリーレポートをクリック。

デイリーレポートの機能説明

![デイリーレポート円グラフ_R.png](https://www.temairazu.com/net/manual/img/デイリーレポート円グラフ_R.png)

①前日・当日に予約、またはキャンセルされた室数、人数、総売上を表示します。

②確認したい日付を選択することができます。デフォルトは当日が表示されます。

③予約日ベースで各種集計結果を表示します。

④表示させた数値やグラフ等のデータをPDFファイルとしてダウンロードできます。

⑤歯車アイコンをクリックでグラフを任意の色に変更することができます。

⑥チェックイン日ベースで集計結果を表示します。

⑦ ②で選択した日から1週間の稼働推移を折れ線グラフで確認できます。
