# 短時間予約発生アラート

> id=172 ／ order=161 ／ 出典: 手間いらず Webマニュアル

短時間予約発生アラート

対応バージョン：　手間いらず.ＮＥＴ2
　　　　　　　　　　　TEMAIRAZU YIELD
　　　　　　　　　　　手間いらず自動

【ツール】＞【各種ツール】＞【短時間予約発生アラート】

短時間に予約が急増した際に、在庫コントロール画面にアラートで通知する機能です。
必要に応じて自動で売り止めを行うこともできます。

![短時間予約発生アラート20240703.png](https://www.temairazu.com/net/manual/img/短時間予約発生アラート20240703.png)

①部屋毎に設定されたい場合は「部屋毎に設定」を、全部屋をとおした予約数で設定されたい場合は
　「全部屋で設定」を選択し、「登録」をクリックしてください。
　　　　 
②アラートを表示する条件を設定します。
　時間は5分から60分まで5分間隔で入力できます、0で設定すると無効になります。

③アラート表示のみ、またはアラートを表示のうえ手仕舞いを行うか選択します。いずれかを選択し
　「設定」をクリックしてください。近い期間に対して対象外期間を設けることも可能です。
　オプションの予約情報入力をご利用されている場合に、予約情報入力で登録した予約も
　件数にカウントするか否かをチェックボックスで設定してください。

④手間いらず画面上のアラートだけでなく、メールで通知を行うことも可能です。
　利用する場合はメールアドレスを入力し「登録」をクリックしてください。
　※「,」カンマ区切りで複数のメールアドレスを指定できます

※当機能は一定時間ごとに予約室数を確認し、設定値に達する予約が入っていた場合にアラート表示や
手仕舞いが行われるため、ご予約からアラート表示までに若干のタイムラグが発生する場合があります。
