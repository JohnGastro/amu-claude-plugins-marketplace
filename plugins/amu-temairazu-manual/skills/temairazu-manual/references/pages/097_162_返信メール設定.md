# 返信メール設定

> id=162 ／ order=97 ／ 出典: 手間いらず Webマニュアル

返信メール設定

【設定】＞【基本設定】＞【返信設定】

※こちらの機能をお使いになるには、先にメールアカウント設定と
　返信メールテンプレート設定が完了している必要があります。

![新返信メール設定_R.png](https://www.temairazu.com/net/manual/img/新返信メール設定_R.png)

①基本テンプレートを設定します。
②送信したいサイトにチェックします。
　基本テンプレートとは別にサイト毎に個別で設定したい場合こちらのプルダウンから設定します。
　※個別設定できるサイトは一部に限ります。
　※サイト右側のプルダウンが未選択の場合、基本テンプレートの内容が適用されます。

③宿泊確認メールを、チェックインの何日前の何時に出すか、または何日以内の場合にメール送信しないかを設定します。
④宿泊御礼メールを、チェックアウト後、何日後の何時以降に出すかを設定します。　
⑤最後に登録するボタンをクリックし設定完了です。
