# 予約者検索

> id=177 ／ order=54 ／ 出典: 手間いらず Webマニュアル

予約者検索

【ツール】＞【予約者検索】
【ツール】＞【各種ツール】＞【予約者検索】

・宿泊日や予約サイトなど条件を指定し、予約者を絞り込むことができます。

![予約者検索1_R.png](https://www.temairazu.com/net/manual/img/予約者検索1_R.png)

![予約者検索2_R.png](https://www.temairazu.com/net/manual/img/予約者検索2_R.png)

①予約の種別を選択できます。

②検索する予約サイトを選択できます。

③部屋タイプを絞り込んで検索できます。
　　　
④宿泊日、チェックイン日、チェックアウト日、予約日時、キャンセル日、変更日などを指定して検索や、
　各内容に応じた検索条件を設定できます。
　※フリーワードは予約情報のプラン名、部屋名、備考、詳細に含まれる文言が検索対象となります。

⑤☑をすることでGoToトラベル対象予約を検索することができます。
　プラン名称、 備考、 詳細欄に　goto　または　go to　が部分一致すれば検索されます。
　（大文字小文字を区別しません）

⑥「検索」クリックで上記①～⑤の条件を検索します。

⑦「検索」クリック後、検索条件の画面が閉じます。再表示させたい場合は「検索＋」をクリックします。

⑧「歯車アイコン」クリックでCSV出力項目設定が表示されます。

⑨クリックでCSVダウンロードができます。表計算ソフト等で予約者詳細を閲覧・編集が可能です。
　※詳細はCSV出力機能をご確認ください。
　※⑧⑨ともにオプションのCSV出力機能をお申込みいただいている場合のみご利用いただけます。

⑩検索結果が表示されます。
