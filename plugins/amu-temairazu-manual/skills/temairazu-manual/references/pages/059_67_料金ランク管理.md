# 料金ランク管理

料金ランク管理

【設定】 ＞ 【基本設定】 ＞ 【料金ランク管理】

■料金ランク

![料金ランク管理_R.png](https://www.temairazu.com/net/manual/img/料金ランク管理_R.png)

①料金ランク名を入力し、料金ランクの絞り込みをすることができます。
※スペース区切りで複数キーワードによる検索ができます

②料金ランクを新規作成できます。「追加」をクリックすると下記ウインドウが表示されます。

![料金ランク管理2_R.png](https://www.temairazu.com/net/manual/img/料金ランク管理2_R.png)

②-A 料金ランク名を入力します。
②-B 料金ランクのカラーを指定します。
②-C 「0円登録する」を選択した場合、0円が入力された状態で各プランに料金ランクが追加されます。
　(料金未設定のプランには0円で登録はされず、未設定のままになります。)
　「しない」を選択した場合、空欄で各プランに料金ランクが追加されます。
②-D 「登録」クリックで確定します。

③「並べ替え」をクリックで料金ランクの表示順を変更することができます。下記ウインドウが表示されます。

![料金ランク管理3_R.png](https://www.temairazu.com/net/manual/img/料金ランク管理3_R.png)

③-A 順番を変更したい料金ランクを『ドラッグ＆ドロップ』することで表示順番の並べ替えが可能です。
③-B　「登録」クリックで確定します。

④作成済みの料金ランクの内容を編集したい場合にクリックします。
　この編集画面から料金ランクの使用する・使用しないを選択することも可能です。
※料金ランクを「使用しない」にした場合、料金カレンダー設定において、その料金ランクを設定していたお日にちは
「未設定」の表記になります。（在庫コントロール画面には料金ランクの色が残るため、過去の設定の確認が可能です。）
　また、基本料金カレンダーで「使用しない」料金ランクが使われている月は、部屋毎カレンダーでの設定が行えません。
※料金ランクを「使用しない」にしても、その料金ランクに入力されていた各プランの料金が削除されることはありません。
　　（再度「使用する」に設定すると該当の料金ランクの設定は復活します。）

⑤ 料金ランクのアップロード 画面が表示されます。
　
⑥アイコンマークをクリックすると当該料金ランクを削除できます。
※料金ランクを削除すると、その料金ランクで設定していた料金カレンダー設定がすべて未設定となり、
　その料金ランクに入力されていた各プランの料金がすべて削除されますので、十分ご注意ください。

⑦アイコンマークをクリックすると当該料金ランクを非表示にすることができます。

⑧対象料金ランクのチェックボックスにチェックのうえクリックすると、選択した料金ランクを一括で削除することができます。

⑨対象料金ランクのチェックボックスにチェックのうえクリックすると、選択した料金ランクを一括で非表示にすることができます。
　　

■料金ランク自動作成機能

![料金ランク自動作成機能_R.png](https://www.temairazu.com/net/manual/img/料金ランク自動作成機能_R.png)

既存の料金ランクに設定されている料金に任意の「割引・割増率」を反映し料金ランクを作成できます。
詳細はこちらをご覧ください。

■最低料金・最高料金設定

![最低料金・最高料金_R.png](https://www.temairazu.com/net/manual/img/最低料金・最高料金_R.png)

全プラン共通の最低料金と最高料金の設定が可能です。
※料金の誤入力を防ぐための機能です。
　設定に反した料金について、登録した料金に補正するか、手仕舞いを行うかを選択できます。
　なお、プラン料金が部屋料金（RC）の場合についても、最低料金・最高料金設定は有効になります。
　一人当たりの料金に換算はされませんのでご注意ください。

※プラン毎に最低・最高料金の設定をしている場合（「-」の場合も）には、プラン毎の設定が優先されます。
※ PMSからの料金更新を行っている場合、最低・最高料金の設定を無視した金額で
　更新される場合がありますのでご注意ください。

■プラン手仕舞い優先度設定

![プラン手仕舞い優先度_R.png](https://www.temairazu.com/net/manual/img/プラン手仕舞い優先度_R.png)

・「後で反映した設定を優先させる」にしている場合

詳細料金設定の『プラン手仕舞い』と提供しない曜日・期間(プラン毎の料金設定内)を比較し、
後で設定したものが優先されます。

　　　 例)
　　　　・詳細料金設定でプラン手仕舞いを行う→提供しない曜日・期間の設定を行う
　　　　　→提供しない曜日・期間が優先
　　　　※プラン取得等でプラン情報が更新された際も提供しない曜日が設定されたと判断されます。

　　　　・提供しない曜日・期間の設定を行う→詳細料金設定でプラン手仕舞いを行う
　　　　　→詳細料金設定が優先

・「詳細料金設定を優先させる」にしている場合

提供しない曜日・期間(プラン毎の料金設定内)の設定をされていても詳細料金設定で「プラン手仕舞い」
を行っていない場合は詳細料金設定が優先されます。
