# マスタテンプレート設定

> id=210 ／ order=243 ／ 出典: 手間いらず Webマニュアル

マスタテンプレート設定

【オプション】＞【PMS項目設定】＞【マスタテンプレート設定】

![PMS項目設定マスタテンプレート設定1_R.png](https://www.temairazu.com/net/manual/img/PMS項目設定マスタテンプレート設定1_R.png)

本設定は、出力形式が【2048】のシステムをご利用の施設様が対象となります。
通常は出力されない予約情報を、追加で出力することが可能です。
設定内容については、事前にPMSシステム会社様へご確認のうえ、設定を行ってください。

①で出力先を選択し、②から出力したい項目を選択したうえで、【挿入】をクリックします。
③の枠内に、②で選択した項目が挿入されていることを確認後、【送信】をクリックして設定を完了してください。

なお、③の枠内で入力された改行は自動的に削除されます。
また、①で「備考」を選択した場合は101バイト、
「電文1～4」を選択した場合は256バイトを超えた部分は自動的に削除されます。
