# 予約カード印刷のヘルプ

> id=195 ／ order=226 ／ 出典: 手間いらず Webマニュアル

予約カード印刷のヘルプ

■予約カードが印刷されない

・ログインID・パスワードが正しく入力されているか確認してください。

・予約カード印刷エージェントが起動しているか確認してください。
　（本ツールが起動している場合のみ、予約カードが印刷されます）

・予約カード印刷設定画面で、印刷が「有効」になっているか確認してください。

・印刷対象の宿泊者様のカードステータスが「未印刷」になっているか確認してください。
　【在庫コントロール】＞【宿泊日付】

・チェックアウト日が前日以降の予約のみ印刷されます。

・プリンタの設定に問題がないか確認してください。
　（電源が入っているか、用紙切れがないか等）

・予約カード印刷の契約内容をご確認ください。

※上記を確認しても印刷されない場合は、予約カード印刷エージェントの再起動をお試しください。

■エラー文が表示される

・.NET Frameworkがインストールされていない可能性があります。

・ツールの用紙サイズが設定されていない可能性があります。

■プリンタを買い換えた場合の再設定方法

①印刷ツールを終了します。

②プリンタの入れ替え作業を行います。

③印刷ツール再起動します。

④使用するプリンタを選択します。

⑤設定を保存します。

（移設作業中に印刷される予定だった予約カードは、
プリンタの入れ替え完了後にツールを起動し設定することで、まとめて印刷されます。）

■予約カードを再印刷したい

【在庫コントロール】＞【宿泊日付】
該当宿泊者様のカードステータスの「再印刷」をクリックしてください。

または

【オプション】＞【予約カード印刷設定】＞【再印刷/印刷済設定】
予約サイトや条件を設定し、「設定する」をクリックしてください。
再印刷/印刷済設定の詳細は【予約カード印刷設定】をご覧ください。

■ツールダウンロード時のzip形式・exe形式の使い分け

施設様のセキュリティ設定により、exe形式のファイルをダウンロードできない場合があります。
その場合は、zip形式でダウンロードしてください。

■予約カードへの印字位置を調整したい

予約カード印刷設定のX座標・Y座標を使用して、印字位置を調整してください。
