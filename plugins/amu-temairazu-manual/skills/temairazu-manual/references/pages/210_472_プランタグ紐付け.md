# プランタグ紐付け

> id=472 ／ order=210 ／ 出典: 手間いらず Webマニュアル

プランタグ紐付け

【ツール】＞【各種ツール】＞【ベストレートチェッカー設定】＞【プランタグ紐付け】

自社宿泊予約システムのプランと、各予約サイトに掲載している自施設のプランのうち、
比較対象としたいプランをタグでまとめることで、
ベストレートチェッカーを利用することができます。

![プランタグ紐付け_R.png](https://www.temairazu.com/net/manual/img/プランタグ紐付け_R.png)

①プラン名または部屋タイプを指定して、表示内容を絞り込むことができます。

②各プランごとに、プランタグを選択してください。
　レートサーベイで自施設の設定を行った翌日から設定可能です。
　※プランタグを設定していないプランは、ベストレートチェッカーでの確認はできません。

③「登録する」をクリックしてください。
