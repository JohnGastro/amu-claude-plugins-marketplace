# LOS用売止グループ

LOS用売止グループ

◆LOS用売止グループの新規作成方法
【設定】＞【基本設定】＞【LOS用売止グループ作成】

![los04_R.png](https://www.temairazu.com/net/manual/img/los04_R.png)

![los05_R.png](https://www.temairazu.com/net/manual/img/los05_R.png)

①「追加」をクリックします。
①-(a) 任意の売止グループ名を入力します。
①-(b) LOS連動宿泊数設定 を設定します。
　　　　 LOS連動宿泊数設定の値は、 LOS&ハードルレート宿泊制限設定 のカレンダー画面における最低泊数と連動します。
　　　　例：カレンダー画面で最低泊数が「2」に設定されている場合、
　　　　　　LOS連動宿泊数設定が「1以下」の売止グループに対して売止が設定されます。
①-(c) 売止グループに紐付けるプランを選択し、条件を入力して抽出します。
①-(d) 抽出されたプランが画面下部に表示されますので、紐付けたいプランの右側にある「＋」をクリックします。
①-(e) 「登録」ボタンをクリックします。

②複数のLOS用売止グループを作成した場合、表示順を並べ替えることができます。
③手順①で作成したLOS用売止グループの内容を編集できます。
④削除する場合はゴミ箱アイコンをクリックします。チェックボックスを使用して一括削除も可能です。

◆作成したLOS用売止グループとLOS＆ハードルレートグループの紐付け
【設定】＞【基本設定】＞【LOS＆ハードルレート設定】　

![los06_R.png](https://www.temairazu.com/net/manual/img/los06_R.png)

![los07_R.png](https://www.temairazu.com/net/manual/img/los07_R.png)

⑤売止グループを紐付けたいグループ名の右側にある「売止グループ紐付け」をクリックします。
⑥LOS連動宿泊数設定をもとに、紐付け可能な売止グループを抽出できます。
⑦売止グループ名を入力して対象を絞り込むことができます。
⑧抽出結果が画面下部に表示されますので、紐付けたい売止グループの右側にある「＋」をクリックします。
⑨「登録」ボタンをクリックします。

◆LOS用売止グループの売止状況の確認方法
【設定】＞【基本設定】＞【LOS用売止グループ設定】

![los08_R.png](https://www.temairazu.com/net/manual/img/los08_R.png)

⑩表の高さを任意に変更できます。
⑪表示する日数の選択および移動が可能です。
⑫LOS用売止グループ名をクリックすると、紐付いているサイト・プランを確認できます。
⑬チェックが入っている日が売止状態です。
　本画面は確認専用のため、手動で設定の追加・解除はできません。

◆LOS用売止グループ用のアイコンについて
【設定】＞【予約サイト設定】＞【部屋プラン一覧】

![los10_R.png](https://www.temairazu.com/net/manual/img/los10_R.png)

⑭部屋プラン一覧にアイコンが表示がされます。
　従来の売止グループアイコンとLOS用売止グループのアイコンは形状が異なります。
　アイコンにマウスオーバーすると、該当の売止グループ名が表示されます。

【在庫コントロール画面】

![los11_R.png](https://www.temairazu.com/net/manual/img/los11_R.png)

⑮売止が設定されている日付は、在庫コントロール画面にも表示されます。

■LOS用売止グループの注意事項
・LOS＆ハードルレートグループにLOS用売止グループを紐付けている場合、
　最低泊数に応じて売止グループの設定が更新されます。
　例：最低泊数が「3」の場合 → 2泊以下の売止グループが売止となります。
・売止グループによる手仕舞いは、予約サイトのLOS対応／非対応に関わらず適用されます。
・LOS用売止グループ設定は、LOS & ハードルレートグループに対する設定です。
　宿泊制限確認画面や部屋プラン一覧画面から遷移する、プラン単体の宿泊制限設定は対象外となります。
