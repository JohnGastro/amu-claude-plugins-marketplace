# 日毎でのターゲットプライス設定の変更・削除方法

> id=489 ／ order=196 ／ 出典: 手間いらず Webマニュアル

日毎でのターゲットプライス設定の変更・削除方法

【在庫コントロール画面】＞任意の日付の「全部屋」または各部屋のマスをクリック

![在庫コントロール画面部屋詳細TP2_R.png](https://www.temairazu.com/net/manual/img/在庫コントロール画面部屋詳細TP2_R.png)

①「全部屋詳細設定画面」から、ターゲットプライスの除外設定および削除を行うことができます。
②「各部屋詳細設定画面」から、各部屋ごとの日別ターゲットプライス設定について、
　料金ランクの変更・手仕舞い・削除を行うことができます。
　※ランク変更や手仕舞い、削除を行う場合は、あらかじめ対象日を除外設定で設定したうえで、本操作を行ってください。
　　除外設定を行っていない場合、
　　在庫数や稼働率の変動により、ターゲットプライス設定が再適用され、変更内容は上書きされます。
　
③ターゲットプライスの除外設定と詳細料金設定の両方が設定されている場合は、「TP除外/詳細」と表示されます。
④ターゲットプライスの除外設定がされている場合は、「TP除外」と表示されます。

■全部屋詳細設定画面

![ターゲットプライス設定全部屋詳細設定1_R.png](https://www.temairazu.com/net/manual/img/ターゲットプライス設定全部屋詳細設定1_R.png)

①-1 すべてのターゲットプライスを一括で除外設定できます。
①-2 すべてのターゲットプライスの除外設定を一括で解除できます。
①-3 個別に設定された除外設定を解除できます。
①-4 個別に除外設定を行うことができます。
①-5 登録されている全部屋のターゲットプライス設定を一括で削除できます。

■各部屋詳細設定画面

![ターゲットプライス設定部屋詳細設定2_R.png](https://www.temairazu.com/net/manual/img/ターゲットプライス設定部屋詳細設定2_R.png)

②-1 料金ランクを変更できます。
②-2 手仕舞いを設定できます。
②-3 ターゲットプライス設定を削除できます。
