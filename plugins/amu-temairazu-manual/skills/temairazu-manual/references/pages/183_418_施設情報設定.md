# 施設情報設定

> id=418 ／ order=183 ／ 出典: 手間いらず Webマニュアル

施設情報設定

【ツール】＞【各種ツール】＞【Google連携設定】＞【施設情報登録・編集】

![google施設情報設定.png](https://www.temairazu.com/net/manual/img/google施設情報設定.png)

*は必須入力になります。

①施設様の名称を入力してください。
②施設様の郵便番号を入力してください。（申請する施設様の郵便番号を入力してください）
③国名の選択をしてください。
④都道府県を選択してください。
⑤市区町村を入力してください。
⑥所在地を入力してください。
⑦建物名がある場合には入力してください。
⑧緯度/経度をGoogleマップで確認のうえ入力してください。（Googleマップにない場合はマップ上に最初に設定してください）
⑨電話番号の入力を行ってください。（申請する施設様の番号を入力してください）
⑩メールアドレスの入力を行ってください。（申請する施設様のメールアドレスを入力してください）
⑪WebサイトのURLを入力してください。（申請する施設様を確認できる自社のURLを入力してください）
⑫申請する施設タイプを選択してください。（必ずGoogleマップに登録しているものと同じタイプを選択してください）
