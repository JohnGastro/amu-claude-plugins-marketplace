# プラン一覧ダウンロード

> id=103 ／ order=114 ／ 出典: 手間いらず Webマニュアル

プラン一覧ダウンロード

【ツール】＞【各種ツール】＞【プラン一覧ダウンロード】

手間いらず上のプラン一覧をCSV形式で出力することができます。

![新プラン一覧CSVDL_R.png](https://www.temairazu.com/net/manual/img/新プラン一覧CSVDL_R.png)

出力項目や予約サイトなどを指定して該当するプラン一覧を出力します。

①対象となるプランを指定できます。

②出力ファイル名を選択できます。
　 ※「ファイル名を指定する」を選択した場合、「.csv」以外を変更してください。

③出力する項目を選択できます。

　・管理ID
　　手間いらず内で一意となるIDです。

　・部屋マスタ名称
　　在庫コントロール画面などに表示される手間いらず上で設定した部屋の名称です。

　・プラン名称各
　　予約サイトで設定されているプラン名称です。

　・プランID各
　　予約サイトで設定されているプランのIDです。

　・部屋ID各
　　予約サイトで設定されている部屋のIDです。

　・PMS部屋コード
　　PMS項目設定＞部屋情報で設定されている部屋コードです。
　　※PMS連携機能をご利用のご施設様のみ表示されます。

　・プラングループID
　　プラングループ毎に付与されるIDです。
　　※プラングループ設定をご利用の場合のみ表示されます。

　・プラングループ名称
　　プラングループ一覧で設定したプラングループ名称です。
　　※プラングループ設定をご利用の場合のみ表示されます。

④予約サイトで絞り込みを行います。
　 ※いずれにもチェックがされていない場合は全てのサイトが対象となります。

⑤チェックをした場合、プラングループ順で出力します。

⑥CSVのダウンロードを行います。
