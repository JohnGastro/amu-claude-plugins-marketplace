# 自動売止通知設定

> id=486 ／ order=83 ／ 出典: 手間いらず Webマニュアル

自動売止通知設定

【設定】＞【売止グループ設定】＞【自動売止通知設定】
【設定】＞【基本設定】＞【自動売止通知設定】

自動売止設定が反映された場合、または予約・変更・キャンセル・自動売止の設定の変更等により
自動売止設定が解除された場合、ご施設様が指定したメールアドレスに通知を受け取ることができる機能です。

![自動売止通知設定.png](https://www.temairazu.com/net/manual/img/自動売止通知設定.png)

送信するに☑を入れ、メールアドレスを入力し「登録」をクリックしてください。
