# 予約通知FAX転送設定

> id=77 ／ order=99 ／ 出典: 手間いらず Webマニュアル

予約通知FAX転送設定

・転送設定

【設定】＞【基本設定】＞【予約通知FAX転送設定】

この機能を使って、FAX出力を行う場合、一件当たり30円の通信料金が発生します。

![faxtensou.png](https://www.temairazu.com/net/manual/img/faxtensou.png)

①予約がキャンセルや変更になった場合に転送するか選択できます。

②転送する予約について、予約サイトごとに選択できます。

③転送するFAX番号を入力し、左の「転送する」にチェックを入れ、「登録する」をクリックすれば完了です。

　　※FAX番号を誤入力していた場合であっても、通信料が発生します。
　　※送信元の制限をされている場合「03-6316-1994」からのFAXが受信できるよう設定をお願いします。

④FAXを送信した履歴を確認することが出来ます。
　　※【ツール】＞【各種ツール】＞【FAX送信履歴】からも確認できます。

・FAX送信履歴

【ツール】＞【各種ツール】＞【FAX送信履歴】
または、【設定】＞【基本設定】＞【予約通知FAX転送設定】下部にある【FAX送信履歴】をクリック

予約通知FAX転送設定を設定した場合に、出力されたFAXの履歴を確認することが出来ます。
