# 予約キャンセル機能

> id=252 ／ order=57 ／ 出典: 手間いらず Webマニュアル

予約キャンセル機能

【設定】＞【基本設定】＞【予約詳細画面設定】

一部予約サイトでは、キャンセル情報の取得がエラーなどで失敗してしまうと再取得ができない仕様のため、
予約サイト側でキャンセルになっている予約が、手間いらず管理画面では予約状態のままなどの場合に、
手間いらず上で予約をキャンセル状態にする機能です。
※手間いらず上の予約がキャンセル状態となりますが予約サイト上の予約は更新されません。

下記対象サイトからの予約について、手間いらず管理画面上の表示をキャンセルにすることができます。

■対象サイト
Expedia、Trip.com、Trip.com(New)、一休.com(予約番号が「I（大文字アイ）」で始まる予約を除く）

![新予約キャンセル4_R.png](https://www.temairazu.com/net/manual/img/新予約キャンセル4_R.png)

①予約キャンセル機能を表示するか、表示しないかを設定できます。

![新予約キャンセル3_R.png](https://www.temairazu.com/net/manual/img/新予約キャンセル3_R.png)

②予約詳細より、予約をキャンセルにします。
　※予約詳細の開き方は予約情報管理をご参照ください。
