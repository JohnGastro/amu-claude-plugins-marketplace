# プラングループ一覧

プラングループ一覧

【料金コントロール】＞【プラングループ一覧】

![プラングループ一覧の3_R.png](https://www.temairazu.com/net/manual/img/プラングループ一覧の3_R.png)

①部屋マスタやプラングループ名で検索することができます。

②プラングループを追加できます。
　プラングループ作成方法は概要をご参照ください。

③プラングループ名を変更することができます。

④プラングループに紐づけられているプランを確認できます。

⑤プラングループに紐づいてるプランの料金設定ができます。

![プラングループ料金設定2-01_R.png](https://www.temairazu.com/net/manual/img/プラングループ料金設定2-01_R.png)

　⑤-1　コピー元プラングループの「部屋タイプ」、「プラングループ」を選択すると
　　　　　自動的に該当プラングループの料金が入力されます。
　　　　　コピーした金額に対して割引・割増したい場合は「金額調整方法」を入力していただき、
　　　　　端数が出た際の「端数処理方法」をご選択し「コピーする」をクリックしてください。　
　⑤-2　紐づいているプランの人数区分は黒字で表示されます。
　　　　　グレーアウトしている人数区分のプランは紐づいてはいませんが
　　　　　料金を予め設定しておくことで、その後追加するプランに対象の人数区分のプランがあれば　　　　　　　
　　　　　その料金が自動で反映されます。
　⑤-3　料金管理の入力ミスにより想定外の料金が反映されるリスクを軽減するための機能です。
　　　　　最低料金・・・最低料金で設定した料金より低い料金を設定した場合、最低料金が優先されます。
　　　　　最高料金・・・最高料金で設定した料金より高い料金を設定した場合、最高料金が優先されます。
　　　　　※最低料金・最高料金で設定した料金になった場合、手仕舞いすることもラジオボタンの選択により可能です。
　　　　　また、最低料金・最高料金いずれかの欄に「-」を入力することで、割引割増設定を無効にすることができます。
　⑤-4　提供しない期間、曜日の設定
　　　　　ここで設定した期間、曜日については料金0円で料金更新を行うため、販売されません。
　　　　　※「祝前日」＞「祝日」＞「その他の曜日」の順に設定が優先されます。
　　　　　 　祝日・祝前日が優先されて、手仕舞いがかからなかった場合には、
　　　 　　　祝日扱いになる日を平日（土日含む）扱いに変更することができる【祝日除外設定】から
　　　　　 　設定を行ってください。

⑥プラングループの並べ替えができます。

![プラングループ並べ替え_R.png](https://www.temairazu.com/net/manual/img/プラングループ並べ替え_R.png)

⑦各サイト毎に販売料金を調整することができます。
　クリックして有効にし、割引割増設定を行ってください。
　チェックボックスにチェックをいれ、割引割増設定を入力後、「一括入力」をクリックすると一括入力可能です。

![販売先毎調整_R.png](https://www.temairazu.com/net/manual/img/販売先毎調整_R.png)

⑧プランをプラングループへ紐付けすることができます。
　プラングループ紐付け方法は概要をご参照ください。

⑨作成したプラングループを削除できます。

⑩販売先毎調整を一括で行う場合にクリックします。

⑪デライブドグループ設定を行う場合はクリックして設定を行ってください。
　※デライブドグループ設定についてはこちら

⑫プラングループ毎に表示か非表示に設定することができます。

⑬チェックボックスにチェックを入れると⑫で非表示にしたプラングループを一覧に表示することができます。

⑭チェックボックスにチェックを入れたプラングループを一括で表示か非表示に設定することができます。
　
⑮チェックボックスにチェックを入れたプラングループを一括で削除することができます。
