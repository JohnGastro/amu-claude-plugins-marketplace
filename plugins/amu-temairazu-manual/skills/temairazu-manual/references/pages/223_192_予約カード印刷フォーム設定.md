# 予約カード印刷フォーム設定

> id=192 ／ order=223 ／ 出典: 手間いらず Webマニュアル

予約カード印刷フォーム設定

【オプション】＞【予約カード印刷フォーム設定】
予約カードの印刷フォームを作成できます。
必要に応じて、下記項目を入力してください。

![【新UI】予約カードフォーム設定1_R.png](https://www.temairazu.com/net/manual/img/【新UI】予約カードフォーム設定1_R.png)

①「座標移動」欄でチェックした項目を、指定した座標の値に基づいて移動させます。

②指定した行を削除対象として設定します。

③描画する図形の形状を選択します。

④線の種類を選択します。

⑤線の太さを選択します。

⑥線の色を選択します。

⑦図形の始点のＸ座標（mm単位）を指定します。

⑧図形の始点のＹ座標（mm単位）を指定します。 

⑨図形の終点のＸ座標（mm単位）を指定します。

⑩図形の終点のＹ座標（mm単位）を指定します。

![【新UI】予約カードフォーム設定2_R.png](https://www.temairazu.com/net/manual/img/【新UI】予約カードフォーム設定2_R.png)

⑪「座標移動」欄でチェックした項目を、指定した座標の値に基づいて移動させます。

⑫指定した行を削除対象として設定します。

⑬表示する文字列を登録します。

⑭文字列を配置するＸ座標（mm単位）を指定します。

⑮文字列を配置するＹ座標（mm単位）を指定します。

⑯文字のフォントを指定します。

⑰文字サイズを指定します。 

⑱文字色を指定します。
