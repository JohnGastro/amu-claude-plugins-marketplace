# プランランキング

> id=220 ／ order=259 ／ 出典: 手間いらず Webマニュアル

プランランキング

【オプション】＞【ブッキングアナライザー】＞【プランランキング】

月単位で、各予約サイトの宿泊数に応じた売上順位を確認できます。

![【BA】プランランキング_R.png](https://www.temairazu.com/net/manual/img/【BA】プランランキング_R.png)

①表示する期間と予約サイトを選択します。

②宿泊数・宿泊人数・利用室数・料金の右側にある ▽ をクリックすると、各項目でソートできます。

③手間いらずで取得した予約のうち、予約数の多い順にプランをランキング形式で表示します。
　 ※集計は、予約サイト側のプランIDを基準に行われます。

④集計結果は、CSV形式またはPDF形式でダウンロードできます。
