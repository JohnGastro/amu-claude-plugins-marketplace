# 一括販売停止機能

> id=553 ／ order=119 ／ 出典: 手間いらず Webマニュアル

一括販売停止機能

【ツール】＞【各種ツール】＞【一括販売停止】

![一括販売停止_01.png](https://www.temairazu.com/net/manual/img/一括販売停止_01.png)

サイト毎に期間の指定をせず、料金在庫管理期間中の販売を停止できる機能です。下記のようなシーンでご活用ください。

・料金の見直しをする間に予約が入らないようにしたい
・予約サイト側でプロモーション設定の見直しをするため、一旦全日程の販売を止めたい
・手間いらずメンテナンス時、特定の予約サイト以外からは予約が入らないようにしたい　など

※予約サイト更新設定で更新が無効となっているサイトに対しては、本機能での販売停止ができません。
※本機能での販売停止は、予約サイト毎にご設定いただいている料金在庫管理期間内にのみ有効です。

＜在庫コントロール画面の表示＞
在庫コントロール表の右上、予約サイト更新設定の設定状況の横に一括販売停止の設定状況が表示されます。

![一括販売停止_02.png](https://www.temairazu.com/net/manual/img/一括販売停止_02.png)

販売中・・・本機能での販売停止をしていない状態
停止中・・・本機能で全予約サイトを販売停止している状態
一部停止中・・・本機能で一部の予約サイトを販売停止している状態（オンマウスで停止中の予約サイトが表示されます）
