# 施設設定

施設設定

【ツール】＞【各種ツール】＞【レートサーベイ設定】＞【施設設定】

本機能の対応サイトに登録されている施設の中から、表示したい施設を選択して設定します。
施設は、自施設を除き最大15施設まで設定できます。
対応サイトの詳細については、こちらをご確認ください。

・歯車マーク

![歯車マーク.png](https://www.temairazu.com/net/manual/img/歯車マーク.png)

＞追加施設設定
Booking.comを利用している施設を追加する場合、デフォルトでは「1泊販売している施設」が検索・表示されます。
「2泊販売している施設」を検索したい場合は、「2泊」を選択し、「登録」をクリックしてください。
※現在は、「1泊」または「2泊」販売の施設のみを検索対象としています。

・対象施設の追加

![新UI_レートサーベイ01_2.png](https://www.temairazu.com/net/manual/img/新UI_レートサーベイ01_2.png)

①選択している施設数が上限に達していない場合、対象施設を追加できます。

![レートサーベイ自動施設設定②_R.png](https://www.temairazu.com/net/manual/img/レートサーベイ自動施設設定②_R.png)

②施設名を入力して、対象施設を検索できます。

③エリアを指定して、対象施設を絞り込むことができます。

④クリックすると、②・③で指定した条件に基づいて検索結果を表示します。

![レートサーベイ自動施設設定③_R.png](https://www.temairazu.com/net/manual/img/レートサーベイ自動施設設定③_R.png)

　
⑤登録したい施設名の左側にあるチェックボックスにチェックを入れて選択します。

⑥予約サイトのユーザー画面へ遷移します。
　　
⑦本機能対応サイトと連携している場合、各予約サイトに掲載されている自施設の情報が表示されます。
※各予約サイトのアカウント設定で連携情報が設定されている必要があります。

⑧「登録」をクリックすると、⑤で選択した施設がレートサーベイの対象施設として登録され、
　⑦で表示されている予約サイトに掲載中の自施設も併せて登録されます。

・施設設定

![レートサーベイ自動施設設定④_R.png](https://www.temairazu.com/net/manual/img/レートサーベイ自動施設設定④_R.png)

①並び替えができます。（自施設は常に最上段に表示されます）

②予約サイトのユーザー画面へ遷移します。

③対象施設から除外できます。
　また、「一括削除」をクリックすると、複数の施設をまとめて対象施設から除外できます。
