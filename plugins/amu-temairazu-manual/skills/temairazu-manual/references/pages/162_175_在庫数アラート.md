# 在庫数アラート

> id=175 ／ order=162 ／ 出典: 手間いらず Webマニュアル

在庫数アラート

対応バージョン：　手間いらず.ＮＥＴ2
　　　　　　　　　　　 TEMAIRAZU YIELD
　　　　　　　　　　　 手間いらず自動

【ツール】＞【各種ツール】＞【在庫数アラート】

在庫コントロール画面の残室が、設定した各在庫数以下になった場合にアラートを表示する機能です。
アラートをメールで通知することもできます。
※予約取得や在庫手動変更などにより設定した在庫数以下になったタイミングで表示・通知されます。

![在庫数アラート22.png](https://www.temairazu.com/net/manual/img/在庫数アラート22.png)

①部屋毎か全部屋で設定するか選択できます。

②手間いらずの部屋マスタごとに、最大で3つまで在庫数の設定をすることができます。
　必ず　在庫数1＞在庫数2＞在庫数3　の順となるように、整数で設定してください。
　※在庫数に空欄または「0」を設定した場合、設定が無効となります。

③アラート対象外期間を直近の期間で設定できます。

④アラート対象外期間を任意の期間で指定して設定できます。

⑤手間いらず画面上のアラートに加え、メールでの通知も行うことができます。
　送信先メールアドレスを入力し、「登録」をクリックしてください。
　※「,」カンマ区切りで複数のメールアドレスを指定できます
