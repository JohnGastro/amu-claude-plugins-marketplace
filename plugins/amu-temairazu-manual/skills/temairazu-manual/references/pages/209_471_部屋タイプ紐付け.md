# 部屋タイプ紐付け

> id=471 ／ order=209 ／ 出典: 手間いらず Webマニュアル

部屋タイプ紐付け

【ツール】＞【各種ツール】＞【ベストレートチェッカー設定】＞【部屋タイプ紐付け】

各予約サイトに掲載している自施設の部屋タイプと、
手間いらずの「部屋プラン一覧」で設定している部屋を紐付けることで、
ベストレートチェッカーを利用することができます。

![部屋タイプ紐付け_R.png](https://www.temairazu.com/net/manual/img/部屋タイプ紐付け_R.png)

①キーワードを指定して、表示内容を絞り込むことができます。

②部屋タイプの紐付けを行ってください。
　※「部屋プラン一覧」で設定している部屋の紐付けと完全に一致させる必要があります。
　※レートサーベイで自施設の設定を行った翌日から設定可能です。
　※部屋タイプの紐付けを行っていないサイトの部屋については、ベストレートチェッカーでの確認はできません。
　※誤った部屋を紐付けた場合、正確な料金は表示されません。
　　正しい部屋に紐付け直し、画面を更新すると、正確な料金が表示されます。

③「登録する」をクリックしてください。
