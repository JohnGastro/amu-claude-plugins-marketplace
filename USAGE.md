# 使い方マニュアル

このプラグインは、TEMAIRAZU（手間いらず）の操作方法をClaudeに聞くための社内用マニュアル検索プラグインです。

## できること

- 在庫調整の手順を確認する
- 売止め・手仕舞いの操作を確認する
- 料金、料金ランク、プラン料金設定を確認する
- 予約確認、予約変更、キャンセル関連の操作を確認する
- 部屋タイプ、プラン、予約サイト設定の手順を確認する
- 予約サイト連携まわりの設定箇所を探す

## 聞き方

普通に質問できます。

```text
手間いらずで在庫を売止めする方法を教えて
```

```text
料金ランクをアップロードする手順は？
```

```text
予約キャンセル機能はどこから使う？
```

```text
楽天の部屋タイプ紐付けを確認したい
```

明示的にプラグインを使いたいときは `/temairazu` を付けます。

```text
/temairazu じゃらんのプラン設定を確認したい
```

## 回答の見方

Claudeは基本的に以下の形で答えます。

```md
手順:
1. ...
2. ...
3. ...

確認ポイント:
- ...

参照:
- ...
```

`参照` には、根拠にしたマニュアルページ名が入ります。

## 質問のコツ

画面名ややりたい作業を入れると探しやすくなります。

良い例:

```text
手間いらずで日別に料金ランクを変更する方法
```

```text
予約サイトごとの売止グループ設定を確認したい
```

```text
プラン料金設定のダウンロード方法
```

曖昧な例:

```text
料金どうする？
```

曖昧な場合でもClaudeが候補を出しますが、具体的な画面名や作業名がある方が正確です。

## 注意

- このプラグインは操作マニュアルを参照するもので、施設ごとのログイン情報や契約情報は含みません。
- マニュアルにない内容は確定情報として扱わないでください。
- 実際の画面がマニュアルと違う場合は、TEMAIRAZUのサポートまたは管理者に確認してください。
