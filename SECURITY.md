# 取り扱い注意

このリポジトリはAmu社内利用を前提としています。

## private repoで運用する

TEMAIRAZU（手間いらず）のログイン後マニュアル本文を含むため、public repoにはしないでください。

## 共有範囲

共有先は、Amu社内でTEMAIRAZUの操作が必要なメンバーに限定してください。

避けること:

- public GitHubへの公開
- 社外メンバーへの招待
- 外部公開NotionやWebページへの転載
- マニュアル全文のSlack等への無制限貼り付け

## 含まれていないもの

- 施設ログインID
- パスワード
- 施設固有の予約情報
- 顧客情報

## 画像について

画像ファイルは同梱していません。Markdown内ではTEMAIRAZU公式側の公開画像URLを参照しています。

## 問題が起きた場合

公開範囲を誤った場合は、すぐにGitHub repoをprivate化または削除し、関係者に連絡してください。
